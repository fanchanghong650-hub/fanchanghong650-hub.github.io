五、Hexo Front Matter

每篇 Hexo 文章开头必须包含 Front Matter（用 --- 包裹的 YAML 块）：

支持的字段

字段 必填 说明
title 是 文章标题
date 是 发布日期，格式 YYYY-MM-DD HH:mm:ss
categories 否 分类，支持多级：[一级, 二级]
tags 否 标签列表：[tag1, tag2]
mathjax 否 设为 true 以启用数学公式
updated 否 更新日期
description 否 文章摘要
top_img 否 文章顶部大图
cover 否 文章封面图
comments 否 设为 false 禁用评论
