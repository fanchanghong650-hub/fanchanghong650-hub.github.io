用 AI 优化空洞骑士钢魂速通路线——从 3 小时到 90 分钟

发表于2026-05-31|更新于2026-05-31|游戏
|浏览量:

最近想重打苦痛之路，于是开了个钢魂档。作为一个辐辉级全通（除五门）的老玩家，我想找一条最快的路线冲到白色宫殿。于是让 Claude Code 帮我来了一轮深度优化。

1. 背景 & 目标

条件 说明
模式 钢魂模式（一命通关）
目标 最快抵达白色宫殿入口
禁用 劈吼、shade skip 等 bug/glitch
允许 quit-out 回长椅（速通常规操作）
偏好 沿途拿苍白矿石强化骨钉，不收集低语之根

我自己原本的流程大概是：

预估 2.5-3 小时左右才能到白宫。

2. 核心发现：白色宫殿的真正门槛

Claude 的首轮研究直接点出了一个我忽视的事实：

白色宫殿只需要 觉醒梦之钉（1800 精华）+ 帝王之翼，除此之外的所有区域——王国边缘、蜂巢、深巢、皇后花园——都不是必须的。

我原来的路线在拿到帝王之翼之后，又跑了 四个大区域，加起来要多花 60-75 分钟，而这些区域的产出（国王印记、蜂巢之血、电车通行证、白色碎片）对进入白宫没有任何帮助。

3. 精华路线的重新设计

3.1 白色防御者是隐藏 MVP

研究过程中最大的惊喜：白色防御者给 300 精华，而且可以重复挑战。这直接改变了整个精华收集策略。

不再需要：无眼（200）、戈布（100）、呼啸悬崖全区域、所有低语之根。

3.2 四个梦境 Boss 全在必经之路上

梦境 Boss 精华 位置 备注
失落近亲 400 古老盆地 破碎容器尸体旁
失败冠军 300 遗忘十字路 假骑士尸体旁
灵魂暴君 300 泪水之城 灵魂大师尸体旁
白色防御者 300 × N 皇家水道 可重复挑战
小计 1300+ 全部零绕路

梦境 Boss 死亡不删档——对钢魂来说这是巨大的安全网。

3.3 精华账本

一个低语之根都不用。无眼、戈布、呼啸悬崖全部跳过。

4. 最终路线

第一阶段：能力 Rush

  • 假骑士 → 复仇之魂，先祖土墩存档
  • 大黄蜂 → 冲刺，退出回十字路
  • 买易碎力量（钢魂永久），螳螂爪，跳过大螳螂
  • 灵魂大师 → 下砸，骨钉强化 #1
  • 买光蝇灯笼，水晶山峰 → 水晶之心 + 黑砸 → 坠落安息之地
  • 梦之钉 + 同房间 Xero（100 精华）

第二阶段：帝王之翼 + 梦境 Boss 连战

  • 古老盆地：破碎容器 → 帝王之翼，梦钉尸体 → 失落近亲（400），苍白矿石 #1
  • 鹿角回十字路 → 失败冠军（300），先知处苍白矿石 #2
  • 梦门放先知处
  • 泪水之城：灵魂暴君（300）
  • 水晶山峰圣巢之冠 → 苍白矿石 #3
  • 骨钉匠：螺旋骨钉（伤害 17 + 易碎力量 = 有效伤害 25.5）
  • 真菌荒地：长老胡（100）

第三阶段：酸泳 + 补满精华

  • 皇家水道：粪虫防御者 → 伊斯玛的眼泪
  • 下砸破地 → 白色防御者 #1（300） → #2（补最后 300）
  • 梦门传送 → 先知 → 觉醒梦之钉
  • 古老盆地 → 宫殿广场 → 进入白宫 🎉

5. 优化对比

维度 原路线 优化后
大黄蜂 #2 ✔ ✘ 跳过
蜂巢 ✔ ✘ 跳过
呼啸悬崖（戈布） ✔ ✘ 跳过
无眼 ✔ ✘ 跳过
低语之根 若干 零个
大螳螂 ✔ ✘ 跳过
苍白矿石 未规划 3-4 块 → 螺旋骨钉
预计时间 ~160-190 min ~90-110 min

6. 几个关键决策的理由

为什么跳过大螳螂？ 骄傲印记对白宫平台跳跃不是刚需。省 3 分钟。

为什么买灯笼而不是下砸进水晶山峰？ 水晶山峰暗房需要灯笼才能拿黑砸，虽然老玩家可以摸黑走，但有灯笼更舒服。

为什么白色防御者替代无眼？ 去苍绿之径石之庇护所本身就是绕路，而皇家水道是去拿酸泳的必经之路。白色防御者就在粪虫防御者房间的下方——零额外移动。

为什么不用低语之根？ 给的少，过程无聊，还得记位置。用可以重复挑战的白色防御者来补精华干净得多。

7. AI 辅助路线优化的体会

整个过程最令人印象深刻的是 Claude Code 的 workflow 多代理模式：

  1. 研究阶段：并行搜索 B站、YouTube、Speedrun.com、Steam 社区指南、Hollow Knight Wiki
  2. 交叉验证：不同来源的路线互相对比（比如速通社区的 NMG 路线 vs Steam 社区的钢魂安全路线）
  3. 合成优化：基于我的个人偏好（辐辉级实力、不收集低语之根、想要苍白矿石）重新设计路线

我原本以为自己作为全辐辉玩家对游戏的理解已经足够深，但 AI 帮助我看到了 地理效率——哪些区域在特定目标下是冗余的——以及 白色防御者作为可重复精华源 这个被忽视的选项。

从 3 小时的流程压到 90 分钟，这个优化幅度超出了我的预期。
