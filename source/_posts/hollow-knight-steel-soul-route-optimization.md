---

## 1. 背景 & 目标

| 条件 | 说明 |
|------|------|
| **模式** | 钢魂模式（一命通关） |
| **目标** | 最快抵达白色宫殿入口 |
| **禁用** | 劈吼、shade skip 等 bug/glitch |
| **允许** | quit-out 回长椅（速通常规操作） |
| **偏好** | 沿途拿苍白矿石强化骨钉，不收集低语之根 |

我自己原本的流程大概是：

```
十字路 → 苍绿之径 → 真菌荒地 → 泪水之城 → 水晶山峰 → 安息之地
→ 古老盆地 → 皇家水道 → 王国边缘 → 蜂巢 → 深巢 → 皇后花园
→ 全图打梦境Boss凑精华 → 白色宫殿
```

预估 **2.5-3 小时**左右能到白宫。

---

## 2. 白色宫殿的真正门槛

> 白色宫殿只需要 **觉醒梦之钉（1800 精华）+ 帝王之翼**，除此之外的所有区域——王国边缘、蜂巢、深巢、皇后花园——都不是必须的。

---

## 3. 精华路线的重新设计

### 3.1 白色防御者是隐藏 MVP

研究过程中最重要的发现：**白色防御者给 300 精华**，而且可以重复挑战。这直接改变了整个精华收集策略。

### 3.2 四个梦境 Boss 全在必经之路上

| 梦境 Boss | 精华 | 位置 | 备注 |
|-----------|------|------|------|
| 失落近亲 | 400 | 古老盆地 | 破碎容器尸体旁 |
| 失败冠军 | 300 | 遗忘十字路 | 假骑士尸体旁 |
| 灵魂暴君 | 300 | 泪水之城 | 灵魂大师尸体旁 |
| 白色防御者 | 300 × N | 皇家水道 | 可重复挑战 |
| **小计** | **1300+** | | 全部零绕路 |

梦境 Boss 死亡不删档——对钢魂来很安全。

### 3.3 精华账本

```
Xero（安息之地同房间）       100
失落近亲（古老盆地）         400
失败冠军（遗忘十字路）       300
灵魂暴君（泪水之城）         300
长老胡（真菌荒地）           100
白色防御者 #1（皇家水道）     300
白色防御者 #2~3             300
────────────────────────────
总计                        1800  ← 精确到位
```

一个低语之根都不用。无眼、戈布、呼啸悬崖全部跳过。

## 4. 最终路线

### 第一阶段：能力 Rush

```
十字路 → 苍绿之径 → 真菌荒地 → 泪水之城 → 水晶山峰 → 安息之地
```

- 假骑士 → 复仇之魂，**先祖土墩存档**
- 大黄蜂 → 冲刺，**退出回十字路**
- **买易碎力量（钢魂永久）**，螳螂爪，**跳过三螳螂**
- 灵魂大师 → 下砸，骨钉强化 #1
- **买光蝇灯笼**，水晶山峰 → 水晶之心 + 黑砸 → 坠落安息之地
- 梦之钉 + **同房间 Xero（100 精华）**

### 第二阶段：帝王之翼 + 梦境 Boss 连战

- 古老盆地：破碎容器 → 帝王之翼，梦钉尸体 → **失落近亲（400）**，苍白矿石 #1
- 鹿角回十字路 → **失败冠军（300）**，先知处苍白矿石 #2
- **梦门放先知处**
- 泪水之城：**灵魂暴君（300）**
- 水晶山峰圣巢之冠 → 苍白矿石 #3
- 骨钉匠：**螺旋骨钉**（伤害 17 + 易碎力量 = 有效伤害 25.5）
- 真菌荒地：**长老胡（100）**

### 第三阶段：酸泳 + 补满精华

- 皇家水道：粪虫防御者 → 伊斯玛的眼泪
- 下砸破地 → **白色防御者 #1（300）** → **#2（补最后 300）**
- **梦门传送** → 先知 → 觉醒梦之钉
- 古老盆地 → 宫殿广场 → **进入白宫** 🎉
