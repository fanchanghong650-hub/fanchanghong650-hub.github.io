## 为什么第二章是整本书的"灵魂"

如果你只能认真读《Introduction to Probability》(Blitzstein & Hwang) 的一个章节，读第二章。

这不是夸张。第一章讲的是"概率是什么"（计数、概率空间公理），是对概率论基本语言的入门。第三章开始讲随机变量，那是工具层面的展开。但第二章讲的是**概率论的思维方式本身**。

Blitzstein 在哈佛 Stat 110 的第一堂课上就说了一句贯穿整个学期的话：

> **"Conditioning is the soul of statistics."**（条件是统计学的灵魂。）

这句话不是修辞。你学完第二章之后回头看，会发现本书后面的几乎每一个重要定理——LOTUS、迭代期望律、贝叶斯推断、马尔可夫链——背后都是"以什么为条件"这个问题。

因此，对第二章采用 2/8 法则：**找出最核心的 20% 概念，它们贡献了 80% 的理解**。下面是这 20%。

## 一、条件概率的定义：一切从这里开始

### 1.1 定义

$$
P(A \mid B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0
$$

这个公式看起来平淡无奇。但它的**含义**才是关键：

**$P(A \mid B)$ 是"在已经知道 $B$ 发生的前提下，$A$ 发生的概率"。**

而这句话的几何含义是：**你把样本空间从 $S$ 缩小到了 $B$，然后在新的空间里重新衡量 $A$ 的比例。**

用你学线性代数的语言来说：**条件是概率论中的"基变换"**。当你对 $B$ 条件化时，$B$ 变成了你的新全集，所有概率都相对于 $B$ 重新标度。这和你换了一组基底后重新表达向量坐标是同一个思想。

### 1.2 为什么这样定义？

几个关键直觉：

1. **$B$ 已经是事实。** 你已经知道 $B$ 发生了，所以你不再关心 $S$ 中 $B$ 之外的那部分。你只关心 $A$ 中落在 $B$ 内部的部分，即 $A \cap B$。

2. **分母是归一化。** $P(B)$ 出现在分母，是为了让 $B$ 本身的概率变成 1：
   $$P(B \mid B) = \frac{P(B \cap B)}{P(B)} = 1$$
   这保证 $P(\cdot \mid B)$ 在 $B$ 上仍然是一个合法的概率测度（满足三条公理）。

3. **乘法定理。** 把定义改写为 $P(A \cap B) = P(A \mid B)P(B) = P(B \mid A)P(A)$。这是概率论中最常用的"串联"公式——计算两个事件同时发生的概率，可以分解为"第一个发生"乘以"在第一个发生的前提下第二个发生"。

### 1.3 为什么说它如此重要？

没有条件概率，概率论就只是一个计数器。有了条件概率，概率论变成了**一个能处理信息和不确定性的推理系统**。

你获得了一条新信息（$B$ 发生了）——你对世界的信念应该如何更新？答案：把 $P(A)$ 换成 $P(A \mid B)$。这个简单的操作，是整个贝叶斯统计、机器学习、信号处理、博弈论的理论基础。

## 二、全概率公式 (LOTP)：分而治之的普遍框架

### 2.1 公式

设 $B_1, B_2, \ldots, B_n$ 是样本空间 $S$ 的一个**划分**（互斥且穷尽），则：

$$
P(A) = \sum_{i=1}^{n} P(A \mid B_i) P(B_i)
$$

### 2.2 直觉：为什么这几乎是一个"免费"的工具？

全概率公式的本质是：**把一个问题切成互斥的几块，分别计算，再按每块的权重加权平均。**

用决策树的思路理解最直观：

```text
        ┌── B1 ── P(A|B1)
        │
   S ───┼── B2 ── P(A|B2)
        │
        └── B3 ── P(A|B3)
```

从树根到 $A$ 有 $n$ 条可能的路径（经过 $B_1$ 再走到 $A$，经过 $B_2$ 再走到 $A$……）。$A$ 的总概率就是每条路径上概率乘积的和：

$$P(A) = P(B_1)P(A \mid B_1) + P(B_2)P(A \mid B_2) + \cdots + P(B_n)P(A \mid B_n)$$

### 2.3 什么时候用"先验"还是"后验"条件概率？

这是很多学生一开始分辨不清的地方。规则很简单：

- **LOTP 的 $B_i$ 通常是"原因"或"假设"**，是你在获得数据之前就有的划分。$P(A \mid B_i)$ 表示"在各种可能的原因下，$A$ 发生的概率是多少"。
- LOTP 的方向是：**从假设到数据**。你枚举所有可能的世界状态，计算每种状态下观察到 $A$ 的概率，再按这些状态的先验概率加权。

### 2.4 关键洞察：LOTP 和贝叶斯定理的对偶关系

LOTP 计算的是 $P(A)$（数据的边际概率），它刚好出现在贝叶斯定理的分母中。这一点后面详细展开。暂时先记住：**LOTP 是在算"证据的整体概率"，而贝叶斯定理是反过来用证据推断原因。**

## 三、贝叶斯定理：翻转条件，更新信念

### 3.1 公式

$$
P(A \mid B) = \frac{P(B \mid A) P(A)}{P(B)}
$$

其中 $P(B) = \sum_i P(B \mid A_i)P(A_i)$（对 $A_i$ 的划分用 LOTP）。

### 3.2 直觉：贝叶斯定理在干什么？

它的本质就一件事：**把条件的"方向"反过来。**

你很容易知道 $P(\text{阳性} \mid \text{有病})$（检测的灵敏度），但你想知道的是 $P(\text{有病} \mid \text{阳性})$（检测阳性后，你真正有病的概率）。这两者完全不同。

贝叶斯定理告诉你如何从前者算出后者：

$$P(\text{有病} \mid \text{阳性}) = \frac{P(\text{阳性} \mid \text{有病}) P(\text{有病})}{P(\text{阳性})}$$

### 3.3 命名每个部分

用条件概率的语言重新标记贝叶斯定理的每一部分：

$$
\underset{\text{后验}}{P(H \mid E)} \;=\; \frac{\overset{\text{似然}}{P(E \mid H)} \;\cdot\; \overset{\text{先验}}{P(H)}}{\underset{\text{证据}}{P(E)}}
$$

- **先验 prior：$P(H)$**——在看到数据之前，你对假设 $H$ 的确信程度
- **似然 likelihood：$P(E \mid H)$**——如果 $H$ 成立，观察到数据 $E$ 的概率
- **证据 evidence：$P(E)$**——数据 $E$ 在所有假设下的整体概率（由 LOTP 计算）
- **后验 posterior：$P(H \mid E)$**——看到数据之后，你对 $H$ 的更新信念

这张图本身就是贝叶斯推断的完整逻辑框架。

### 3.4 一个极其重要的陷阱：检察官谬误 (Prosecutor's Fallacy)

考虑下面这个问题（教材中最经典也最危险的例子之一）：

> 一种罕见病的发病率是 0.1%。检测方法的灵敏度（有病→阳性）是 99%，特异度（没病→阴性）也是 99%。你检测结果为阳性，你真的有病的概率是多少？

大多数人的直觉回答是 99% 或"非常高"。正确的计算：

$$P(\text{有病} \mid \text{阳性}) = \frac{0.99 \times 0.001}{0.99 \times 0.001 + 0.01 \times 0.999} \approx 0.09$$

**原因：** 虽然检测很准，但疾病极其罕见。在 1000 个被检测的人里，大约 1 个是真正有病（且检测阳性），但有大约 10 个没病的人也会被测出阳性（假阳性）。所以阳性的人中，绝大多数是假阳性。

**核心教训：** $P(A \mid B)$ 和 $P(B \mid A)$ 完全不同。混淆两者是人脑最容易犯的概率错误。贝叶斯定理的存在的全部意义，就是让你在数值上正确地做出这个转换。

### 3.5 比率形式：一个让你可以口算贝叶斯定理的版本

如果你看过 3Blue1Brown 关于贝叶斯定理的视频，你可能已经见过这个思想。贝叶斯定理有一个**比率（odds）形式**，它和概率形式在数学上等价，但在直觉上要自然得多——一旦你理解了它，很多贝叶斯问题你甚至可以直接口算。

#### 从概率到比率

在概率形式下，贝叶斯定理需要计算分母 $P(E)$（证据的全概率），这通常是你最不想算的部分——你需要枚举所有假设并用 LOTP 求和：

$$P(H \mid E) = \frac{P(E \mid H) P(H)}{P(E \mid H)P(H) + P(E \mid H^c)P(H^c)}$$

比率形式完全绕过了 $P(E)$。做法是：**同时跟踪假设 $H$ 成立和不成立两种情况，并取它们的比值。**

写出 $H$ 成立时的后验概率：

$$P(H \mid E) = \frac{P(E \mid H) P(H)}{P(E)}$$

再写出 $H^c$（$H$ 不成立）时的后验概率：

$$P(H^c \mid E) = \frac{P(E \mid H^c) P(H^c)}{P(E)}$$

两式相除，$P(E)$ 直接消掉：

$$\frac{P(H \mid E)}{P(H^c \mid E)} = \frac{P(H)}{P(H^c)} \times \frac{P(E \mid H)}{P(E \mid H^c)}$$

这就是贝叶斯定理的**比率形式**：

$$\boxed{\text{后验比率} = \text{先验比率} \times \text{Bayes Factor}}$$

其中每一项的含义：

| 名称 | 公式 | 含义 |
|---|---|---|
| **先验比率 (Prior Odds)** | $\displaystyle O(H) = \frac{P(H)}{P(H^c)}$ | 在看到数据之前，$H$ 成立与不成立的比率 |
| **Bayes Factor (似然比)** | $\displaystyle BF = \frac{P(E \mid H)}{P(E \mid H^c)}$ | 数据 $E$ 在 $H$ 成立时出现的概率，除以它在 $H$ 不成立时出现的概率——也就是**数据的"证据力"** |
| **后验比率 (Posterior Odds)** | $\displaystyle O(H \mid E) = \frac{P(H \mid E)}{P(H^c \mid E)}$ | 看到数据之后，$H$ 成立与不成立的比率 |

最后，如果你想从比率回到概率：

$$P(H \mid E) = \frac{O(H \mid E)}{1 + O(H \mid E)}$$

#### 为什么这个形式更自然？

**因为它的结构直接对应了概率乘法公式。**

先验比率的分子分母，分别乘以 Bayes Factor 的分子分母，就直接得到了后验比率的分子分母：

$$\frac{P(H)}{P(H^c)} \;\times\; \frac{P(E \mid H)}{P(E \mid H^c)} \;=\; \frac{P(E \mid H)P(H)}{P(E \mid H^c)P(H^c)}$$

这无非是在说：**"假设 $H$ 为真的那条路径"和"假设 $H$ 为假的那条路径"，各自乘上数据在该假设下出现的似然，然后比较两条路径的相对大小。** 分母 $P(E)$ 从不需要被计算——它只是在两条路径之间做归一化时才会用到，而在比率视角下，你根本不关心归一化，你只关心两条路径的**相对大小**。

如果你熟悉线性代数，可以这样类比：概率形式像是在计算一个向量的绝对坐标（需要归一化），而比率形式像是在比较两个基向量的系数之比——你不需要知道整个向量的长度。

#### 用比率形式重做罕见病检测（口算版）

回到 3.4 节的罕见病检测问题：

> 发病率 0.1%（1:999）。灵敏度 99%，特异度 99%。阳性后患病概率？

**第一步：写出先验比率。**
- 患病与不患病的比率 = $0.001 : 0.999 = 1 : 999$

**第二步：计算 Bayes Factor。**
- 如果你有病，阳性概率 = 99%（灵敏度）
- 如果你没病，阳性概率 = 1%（假阳性率 = 1 − 特异度）
- Bayes Factor = $\frac{0.99}{0.01} = 99$

**第三步：相乘得到后验比率。**
- 后验比率 = $(1 : 999) \times 99 = 99 : 999 \approx 1 : 10$

这意味着：在阳性的人中，大约每 11 个人里有 1 个是真有病。概率 ≈ $1/11 \approx 9\%$。

**你根本不需要算 $P(E)$。** 分母的 LOTP 求和（$0.99 \times 0.001 + 0.01 \times 0.999$）被完全绕过了。你只需要心算"先验比率 × Bayes Factor"。

#### 再练一个：垃圾邮件过滤器

回到题 2 的垃圾邮件问题：

> 80% 的邮件是垃圾邮件。单词出现率：垃圾邮件 10%，正常邮件 1%。

**第一步：先验比率。**
- 垃圾 : 正常 = $0.8 : 0.2 = 4 : 1$

**第二步：Bayes Factor。**
- "Free money" 在垃圾邮件中出现概率 / 在正常邮件中出现概率 = $0.1 / 0.01 = 10$

**第三步：后验比率。**
- 后验比率 = $(4 : 1) \times 10 = 40 : 1$

转换回概率：$P(\text{垃圾} \mid \text{关键词}) = 40/41 \approx 97.6\%$。和之前用概率形式计算的结果完全一致，但计算路径更加清晰。

#### 核心直觉：Bayes Factor 衡量的是"证据的强度"

在比率形式下，贝叶斯更新被拆解为两个独立的部分：

1. **先验比率**——在没有任何数据的情况下，你对世界的初始信念。它完全由 base rate（基础概率）决定。
2. **Bayes Factor**——数据本身携带的"信息量"。它告诉你，看到数据 $E$ 之后，你应该把对 $H$ 的确信程度**放大或缩小多少倍**。

两者相乘，就是更新后的信念。这个结构干净、模块化，而且非常符合人类直觉——你分别评估"一开始有多信"和"证据有多强"，然后乘起来。

特别是，Bayes Factor 有一个非常直观的解读：

- $BF > 1$：数据 $E$ 支持 $H$（$E$ 在 $H$ 成立时更可能出现）
- $BF = 1$：数据 $E$ 不提供任何信息（$E$ 在两种情况下等可能）
- $BF < 1$：数据 $E$ 反对 $H$（$E$ 在 $H$ 不成立时更可能出现）
- $BF = 99$（像罕见病检测中那样）：数据 $E$ 非常强烈地支持 $H$——但如果先验比率是 1:999，那么即使有这么强的证据，后验比率也只有约 1:10。**强证据可以大幅扭转先验，但不能完全克服极端稀少的 base rate。**

这就是比率形式给你上的最重要的一课：**概率更新 = 先验信念 × 证据强度。** 没有哪一个可以单独决定答案。

## 四、独立性：什么时候条件不起作用？

### 4.1 定义

事件 $A$ 和 $B$ **独立**，当且仅当：

$$P(A \cap B) = P(A) P(B)$$

等价地（当 $P(B) > 0$ 时）：

$$P(A \mid B) = P(A)$$

### 4.2 直觉

独立性的本质是：**知道 $B$ 发生了，不改变你对 $A$ 的概率判断。**

这不是"$A$ 和 $B$ 没有关系"（那是一个模糊的直觉），而是数学上精确的条件：$B$ 的信息对于预测 $A$ **没有价值**。

### 4.3 常犯的错误

1. **互斥 ≠ 独立。** 如果 $A$ 和 $B$ 互斥且概率都非零，则 $P(A \mid B)=0 \neq P(A)$，它们极端**不**独立——$B$ 发生意味着 $A$ 绝不可能。

2. **两两独立 ≠ 相互独立。** 三个事件 $A, B, C$ 两两独立，但有可能 $A \cap B$ 给出关于 $C$ 的信息。一个经典的例子：抛两枚公平硬币：
   - $A = \{\text{第一枚是正面}\}$
   - $B = \{\text{第二枚是正面}\}$
   - $C = \{\text{两枚结果相同}\}$
   
   检查后会发现每对独立（$P(A \cap B) = 1/4 = P(A)P(B)$ 等），但 $P(C \mid A \cap B) = 1$，而 $P(C) = 1/2$——知道 $A$ 和 $B$ 同时发生，完全确定了 $C$。

3. **独立性不能用文氏图的"面积关系"来判断。** 独立性是一种乘积关系，不是几何关系。

### 4.4 一个微妙但重要的结果

如果 $A$ 和 $B$ 独立，那么 $A^c$ 和 $B$、$A$ 和 $B^c$、$A^c$ 和 $B^c$ 也都独立。

证明：$P(A^c \cap B) = P(B) - P(A \cap B) = P(B) - P(A)P(B) = (1-P(A))P(B) = P(A^c)P(B)$。

直觉上：如果知道 $B$ 不改变你对 $A$ 的判断，那么它也不改变你对"非 $A$"的判断。信息不区分正反。

## 五、条件独立：独立性的关键推广

### 5.1 定义

事件 $A$ 和 $B$ **在给定 $C$ 的条件下独立**，当且仅当：

$$P(A \cap B \mid C) = P(A \mid C) P(B \mid C)$$

### 5.2 关键直觉

条件独立意味着：**如果你已经知道了 $C$，那么进一步知道 $B$ 不会给你关于 $A$ 的任何额外信息。**

这比普通独立性更加微妙，但它更贴近实际建模：

- $C$ = 对手的实力（初学者 / 中等 / 高手）
- $A$ = 你赢了第一局棋
- $B$ = 你赢了第二局棋

给定对手的实力 $C$，$A$ 和 $B$ 是条件独立的——因为一旦你知道对手有多强，第一局的结果不提供关于第二局的额外信息。

但 $A$ 和 $B$ **不是无条件独立的**——因为你赢了第一局这件事本身就让你推断对手可能比较弱（给你关于 $C$ 的信息），从而提高了第二局的预期胜率。

### 5.3 条件独立 vs. 无条件独立：互相转化的充要条件

这是条件概率中最容易被低估、但也最深刻的一个问题：**条件独立和完全独立（无条件独立）之间，到底是什么关系？**

短答案是：**两者互不蕴含。** 但真正重要的是埋在这句话下面的那个充要条件——一旦你理解了它，你对"独立性"这个概念的理解就上升了一个层次。

#### 5.3.1 四种可能的排列

在"给定 $C$ 下独立"和"无条件独立"之间，所有四种排列都是可能的：

| 给定 $C$ 下独立？ | 无条件独立？ | 是否可能？ | 典型案例 |
|---|---|---|---|
| ✅ | ✅ | 可能 | $A, B, C$ 完全无关（三个独立硬币分别决定 $A, B, C$） |
| ✅ | ❌ | 可能 | 棋局例子：给定对手实力，两局结果独立；但无条件不独立 |
| ❌ | ✅ | 可能 | 抛两枚硬币，以"两枚结果不同"为条件 |
| ❌ | ❌ | 可能 | 毫无独立性可言 |

前三个案例之所以各自可能，是因为它们背后有本质不同的因果结构。事实上，三个变量之间的关系可以用三种**图模型原型**来分类：

| 图结构 | 名称 | 无条件独立？ | 条件独立（给定 $C$）？ |
|---|---|---|---|
| $A \leftarrow C \rightarrow B$ | 叉子 (Fork) / 共同原因 | ❌ | ✅ |
| $A \rightarrow C \rightarrow B$ | 链 (Chain) | ❌ | ✅ |
| $A \rightarrow C \leftarrow B$ | 对撞子 (Collider) / 共同结果 | ✅ | ❌ |

- **叉子** $A \leftarrow C \rightarrow B$：$C$ 同时影响 $A$ 和 $B$。由于 $A$ 和 $B$ 共享一个共同原因，它们无条件相关（如果你看到 $A$ 发生了，你会推断 $C$ 可能比较大，进而推断 $B$ 也可能比较大）。但一旦你固定了 $C$，这个间接关联就断了——$A$ 和 $B$ 在给定 $C$ 下独立。**棋局例子就属于这一类：对手实力 $C$ 同时影响第一局 $A$ 和第二局 $B$ 的胜负。**

- **对撞子** $A \rightarrow C \leftarrow B$：$A$ 和 $B$ 是独立的原因，$C$ 是它们的共同结果。在这种情况下，$A$ 和 $B$ 无条件独立（两个独立的原因本来就无关），但**给定 $C$ 后**，$A$ 和 $B$ 变得相关——因为如果你知道 $C$ 发生了，而且你知道 $A$ 也发生了，你就会推断 $B$ 可能没有发生（否则 $C$ 就是"过度确定"的了）。5.3.1 节中抛两枚硬币、$C$ 为"结果不同"的例子，正是对撞子结构。

- **链** $A \rightarrow C \rightarrow B$：$A$ 通过 $C$ 影响 $B$。无条件情况下 $A$ 和 $B$ 相关，给定 $C$ 后独立。

> **核心结论：** 条件独立和完全独立之间的转化，完全取决于 $C$ 在图结构中的位置。$C$ 是共同原因（叉子/链）→ 条件独立；$C$ 是共同结果（对撞子）→ 条件相关。

#### 5.3.2 充要条件：从条件独立到无条件独立

假设我们已经知道 $A$ 和 $B$ 在给定 $C$ 下条件独立。问题是：**$A$ 和 $B$ 什么时候也是无条件独立的？**

对于二值事件，答案出奇地简洁。条件独立给出了两个等式：

$$P(A \cap B \mid C) = P(A \mid C)P(B \mid C)$$
$$P(A \cap B \mid C^c) = P(A \mid C^c)P(B \mid C^c)$$

现在计算 $A$ 和 $B$ 的无条件交概率：

$$P(A \cap B) = P(A \cap B \mid C)P(C) + P(A \cap B \mid C^c)P(C^c)$$
$$= P(A \mid C)P(B \mid C)P(C) + P(A \mid C^c)P(B \mid C^c)P(C^c)$$

$A$ 和 $B$ 无条件独立，需要 $P(A \cap B) = P(A)P(B)$。代入 $P(A) = P(A \mid C)P(C) + P(A \mid C^c)P(C^c)$ 和 $P(B)$ 的类似展开，经过代数简化（推导见文末注释），得到：

$$\boxed{[P(A \mid C) - P(A \mid C^c)] \times [P(B \mid C) - P(B \mid C^c)] = 0}$$

这就是充要条件（假设 $0 < P(C) < 1$）。它的含义是：**在条件独立的前提下，$A$ 和 $B$ 无条件独立，当且仅当 $A$ 或 $B$（至少一个）本身与 $C$ 独立。**

换句话说：

- 如果 $P(A \mid C) = P(A \mid C^c)$，即 $A \perp C$，那么即使 $B$ 依赖于 $C$，条件独立 $\implies$ 无条件独立。
- 如果 $P(B \mid C) = P(B \mid C^c)$，即 $B \perp C$，同理。
- 如果 $A$ 和 $B$ **都**依赖于 $C$（即 $C$ 同时影响 $A$ 和 $B$），那么条件独立成立时，$A$ 和 $B$ **必定不是**无条件独立的——这正是叉子结构或棋局例子的情况。

用棋局例子验证：
- $P(W_1 \mid \text{初}) = 0.9$，$P(W_1 \mid \text{中}) = 0.5$，两者不等 → $W_1 \not\perp S$
- $P(W_2 \mid \text{初}) = 0.9$，$P(W_2 \mid \text{中}) = 0.5$，两者不等 → $W_2 \not\perp S$
- $[P(W_1 \mid S) - P(W_1 \mid S^c)] \times [P(W_2 \mid S) - P(W_2 \mid S^c)] \neq 0$
- 所以尽管 $W_1$ 和 $W_2$ 条件独立，它们**不是**无条件独立的。✅

> 这个充要条件的形式结果与 Nogales & Pérez (2017) 的定理一致——他们将该定理推广到了一般随机变量和 Markov 核。对于二值事件的特例，条件简化为上述乘积等式。

#### 5.3.3 反向：从无条件独立到条件独立——以及 XOR 陷阱

反过来，假设 $A$ 和 $B$ 无条件独立。**什么时候它们也在给定 $C$ 下条件独立？**

条件独立需要两个等式同时成立：
$$P(A \cap B \mid C) = P(A \mid C)P(B \mid C) \quad\text{且}\quad P(A \cap B \mid C^c) = P(A \mid C^c)P(B \mid C^c)$$

将无条件独立的约束代入 LOTP 展开后，**同样的乘积条件**
$$[P(A \mid C) - P(A \mid C^c)] \times [P(B \mid C) - P(B \mid C^c)] = 0$$
仍然是一个**必要条件**（即如果条件独立成立，这个乘积必须为零）。但在这个方向上，它**不是充分条件**。

为什么？因为即使 $A \perp C$ 且 $B \perp C$（从而使上述乘积为零），$A$ 和 $B$ 仍然可能在给定 $C$ 下变得相关。最经典的例子是 5.3.1 中抛两枚硬币的对撞子案例：

> $A$ = 第一枚正面（$P(A)=1/2$），$B$ = 第二枚正面（$P(B)=1/2$），两者无条件独立。
> $C$ = $\{HT, TH\}$（两枚结果不同）。
>
> 检验：$P(A \mid C) = 1/2 = P(A \mid C^c)$ → $A \perp C$ ✅
> $P(B \mid C) = 1/2 = P(B \mid C^c)$ → $B \perp C$ ✅
>
> 乘积条件满足，但：$P(A \cap B \mid C) = 0 \neq 1/4 = P(A \mid C)P(B \mid C)$。**条件独立不成立！**

这里的深层原因是：$C$ 虽然和信息 $A$ 个体无关、和信息 $B$ 个体无关，但它与 $(A, B)$ 这个**组合**有关——$C$ 恰好是 $A$ 和 $B$ 的 XOR 函数。知道 $A$ **和** $B$ 同时发生，就可以完全确定 $C^c$。所以当条件于 $C$ 时，$A$ 和 $B$ 被绑定在一起，产生了条件依赖。

**小结（两个方向的对比）：**

| 方向 | 充要条件 | 关键直觉 |
|---|---|---|
| 条件独立 $\to$ 无条件独立 | $[P(A \mid C)-P(A \mid C^c)] \times [P(B \mid C)-P(B \mid C^c)] = 0$ | 只要 $A$ 或 $B$ 之一与 $C$ 独立，条件独立就自动升级为完全独立 |
| 无条件独立 $\to$ 条件独立 | 同一条件是**必要的但不充分的**；还需要排除 XOR 式的组合依赖 | 即使 $A$ 和 $B$ 各自与 $C$ 独立，如果 $C$ 是 $(A, B)$ 的联合函数，条件相关仍然存在 |

#### 5.3.4 图模型视角：为什么这三种结构覆盖了一切？

5.3.1 中的三种图结构（叉子、链、对撞子）并不是巧合。在概率图模型理论中（本书第 7 章之后会详细展开），有一个核心概念叫 **d-分离（d-separation）**：图中的一条路径是否被某个变量集合"阻断"，决定了条件独立的成立与否。

三种结构对应的 d-分离规则：

| 结构 | 路径是否被 $C$ 阻断？ | 结果 |
|---|---|---|
| $A \leftarrow C \rightarrow B$（叉子） | ✅ 阻断（$C$ 是中间节点且被条件化） | $A \perp B \mid C$ |
| $A \rightarrow C \rightarrow B$（链） | ✅ 阻断（$C$ 是中间节点且被条件化） | $A \perp B \mid C$ |
| $A \rightarrow C \leftarrow B$（对撞子） | ❌ **打开**（$C$ 是对撞子，条件化反而打开路径） | $A \not\perp B \mid C$ |

这就是贝叶斯网络中信息流的底层逻辑——也是你将来学图模型时必须掌握的基础。

### 5.4 为什么条件独立是贝叶斯网络的基础

在更大的图景中，条件独立是概率图模型（贝叶斯网络、马尔可夫随机场）的基石。一个有向无环图（DAG）中的每个节点、每条边，本质上都在做一件事：**声明哪些变量在给定哪些变量的条件下独立。**

具体来说，一个贝叶斯网络通过以下声明确定了整个联合分布的结构：

- **每个变量在给定其父节点的条件下，与它的非后代节点独立。**

这正是条件独立的形式定义。5.3 节的图结构（叉子、链、对撞子）是构成这个定义的基本单元。如果你不理解条件独立与无条件独立之间的关系——特别是**为什么条件化一个对撞子会"打开"一条原本独立的路径**——你将来学贝叶斯网络时会反复感到困惑。

理解了这一节，贝叶斯网络就只是一张用箭头画出来的条件独立关系图。

### 题 1：Lewis Carroll 的大理石问题（贝叶斯定理的经典）⭐

**来源：** Stat 110 战略练习 2 (SP2)，原题出自 Lewis Carroll (1893)

> 一个袋子里有一颗大理石，它以等概率是绿色或蓝色的。现在你在袋子里放入一颗绿色大理石，摇匀后随机取出一颗，结果是绿色的。问袋子里剩下的那颗大理石也是绿色的概率是多少？

**▼ 答案与解析**

设事件：
- $G_1$：最初那颗大理石是绿色的
- $G_2$：取出的那颗大理石是绿色的

已知 $P(G_1) = 1/2$。

用贝叶斯定理：

$$P(G_1 \mid G_2) = \frac{P(G_2 \mid G_1) P(G_1)}{P(G_2)}$$

- 如果最初是绿色的，袋子里两颗都是绿色，取出绿色概率 $P(G_2 \mid G_1) = 1$。
- 如果最初是蓝色的，加一颗绿色后，袋子里有一绿一蓝，取出绿色概率 $P(G_2 \mid G_1^c) = 1/2$。
- 用 LOTP：$P(G_2) = 1 \times \frac{1}{2} + \frac{1}{2} \times \frac{1}{2} = \frac{3}{4}$。

所以：

$$P(G_1 \mid G_2) = \frac{1 \times 1/2}{3/4} = \frac{2}{3}$$

**剩下一颗也是绿色的概率是 2/3。**

直觉：取出绿色更可能发生在"最初也是绿色"的世界里，所以条件于取出绿色后，最初是绿色的概率从 1/2 升到了 2/3。

### 题 2：垃圾邮件过滤器（贝叶斯定理的应用）⭐

**来源：** Stat 110 战略练习 2 (SP2)，3.2

> 80% 的邮件是垃圾邮件。单词"Free money"出现在 10% 的垃圾邮件中，以及 1% 的非垃圾邮件中。如果一封邮件包含"Free money"，它是垃圾邮件的概率是多少？

**▼ 答案与解析**

设 $S$ 表示邮件是垃圾邮件，$W$ 表示包含"Free money"。

已知：$P(S) = 0.8$，$P(W \mid S) = 0.1$，$P(W \mid S^c) = 0.01$。

用贝叶斯定理：

$$P(S \mid W) = \frac{P(W \mid S)P(S)}{P(W \mid S)P(S) + P(W \mid S^c)P(S^c)}$$

$$= \frac{0.1 \times 0.8}{0.1 \times 0.8 + 0.01 \times 0.2} = \frac{0.08}{0.08 + 0.002} = \frac{0.08}{0.082} \approx 0.9756$$

**答案：约 97.6%。**

**洞察：** 即便"Free money"在垃圾邮件中的出现率只有 10%（看起来不算很高），但由于这个词在正常邮件中极少出现，以及垃圾邮件的基数高，所以看到这个词后，垃圾邮件的概率飙升。

### 题 3：罕见病检测问题（检察官谬误）⭐⭐

**来源：** 教材 Chapter 2 例题

> 某疾病发病率 1%。检测的真阳性率 95%，假阳性率 5%。你检测阳性，你真的有病的概率是多少？

**▼ 答案与解析**

设 $D$ 表示有病，$+$ 表示检测阳性。

已知 $P(D) = 0.01$，$P(+ \mid D) = 0.95$，$P(+ \mid D^c) = 0.05$。

$$P(D \mid +) = \frac{0.95 \times 0.01}{0.95 \times 0.01 + 0.05 \times 0.99} = \frac{0.0095}{0.0095 + 0.0495} \approx 0.161$$

**只有大约 16%！** 虽然检测看起来很准，但因为疾病极其罕见，假阳性的人数远远超过真阳性。

**核心教训：** $P(+ \mid D)$ 和 $P(D \mid +)$ 是完全不同的量。永远不要不经过贝叶斯定理就直接将似然当作后验。

### 题 4：证据与犯罪嫌疑（辛普森悖论的雏形）⭐⭐

**来源：** Stat 110 战略练习 2 (SP2)，3.3

> 有两项证据 $E_1$ 和 $E_2$。
> (a) 能否设计一个场景，使得 $E_1$ 单独增加嫌疑，$E_2$ 单独也增加嫌疑，但 **两者同时出现反而降低嫌疑**？
> (b) 证明：逐次更新（先看到 $E_1$，再在更新后的概率上看到 $E_2$）和一次性更新（同时看到 $E_1$ 和 $E_2$）得到相同的后验概率。

**▼ 答案与解析**

**(a)** 可以。关键是要理解：两份证据的**组合模式**可以携带单独证据所没有的信息。下面是一个具体的数值例子。

**场景设定：** 100 个嫌疑人，先验有罪概率为 30%（30 人有罪，70 人无辜）。

- $E_1$ = 嫌疑人鞋底有案发现场的泥土
- $E_2$ = 嫌疑人衣服上有案发现场的纤维

两份证据在有罪者和无辜者身上的分布**不同**：

| 证据组合 | 有罪的人 (30人) | 无辜的人 (70人) |
|---|---|---|
| 只有 $E_1$（泥土，无纤维） | 12 人 (40%) | 7 人 (10%) |
| 只有 $E_2$（纤维，无泥土） | 12 人 (40%) | 7 人 (10%) |
| $E_1$ 和 $E_2$ 都有 | 4.5 人 (15%) | 21 人 (30%) |
| 都没有 | 1.5 人 (5%) | 35 人 (50%) |

这个分布背后的直觉：有罪者作案时往往只会留下一种痕迹（要么泥土要么纤维），很少两者都留下；而无辜者中，泥土和纤维经常一起出现——比如因为他们都是某个特定场所（如工作单位）的常客，泥土和纤维都来自那里，而非案发现场。

现在验证数学：

**只看 $E_1$（泥土）：** 有 $E_1$ 的人 = 有罪者中有 $E_1$ 的 (12 + 4.5) + 无辜者中有 $E_1$ 的 (7 + 21) = 44.5 人。其中有罪的 16.5 人。

$$P(\text{有罪} \mid E_1) = \frac{16.5}{44.5} \approx 37.1\% > 30\% \quad \text{（嫌疑上升）}$$

**只看 $E_2$（纤维）：** 对称的计算，同样 ≈ 37.1% > 30%。**（嫌疑上升）**

**同时看到 $E_1$ 和 $E_2$：** 同时有两份证据的人 = 4.5 + 21 = 25.5 人。其中有罪的 4.5 人。

$$P(\text{有罪} \mid E_1, E_2) = \frac{4.5}{25.5} \approx 17.6\% < 30\% \quad \text{（嫌疑大幅下降！）}$$

**为什么会这样？** 注意第四列的关键数字：无辜者中同时拥有两份证据的比例（30%）远高于有罪者（15%）。单独看一份证据时，这份证据在有罪者中更常见（55% vs 40%），所以嫌疑上升。但当你要求**两份证据同时出现**时，这个组合在无辜者中反而更典型——泥土和纤维在无辜者中高度正相关（它们有共同的来源），而在有罪者中不那么相关。

这就是**辛普森悖论在贝叶斯推断中的体现**：每个单独维度上的趋势，在合并数据时发生了反转。它提醒我们——**永远不能只看证据的单独效应，必须看证据组合的联合分布。**

> #### 🔗 连接经典：辛普森悖论的两种面孔
>
> 如果你听过 Blitzstein 在 Stat 110 课堂上讲的辛普森悖论经典案例，你可能会认出这道题和它是同一个数学结构。那个经典例子是：
>
> | | 心脏手术 | 贴创口贴 | 总成功率 |
> |---|---|---|---|
> | **好医生** | 90%（做了 100 台） | 100%（做了 10 个） | 90.9%（100/110） |
> | **差医生** | 80%（做了 10 台） | 95%（做了 100 个） | 93.6%（103/110） |
>
> 好医生在**每种手术类型**上的成功率都高于差医生（90% > 80%，100% > 95%），但**总体**成功率反而更低（90.9% < 93.6%）。
>
> 原因：好医生大部分时间在做困难的心脏手术，差医生大部分时间在做简单的创口贴。**分组权重不同导致了反转。**
>
> 这两种面孔的对应关系：
>
> | 经典版本（频率表） | 本题版本（贝叶斯更新） |
> |---|---|
> | 好医生 | 有罪的人（$H$ 成立） |
> | 差医生 | 无辜的人（$H$ 不成立） |
> | 心脏手术（困难任务） | 只有一份证据的情形 |
> | 创口贴（简单任务） | 两份证据同时出现的情形 |
> | 好医生多做困难手术 | 有罪者中两份证据同时出现比例低（15% vs 30%） |
> | 每个子类别好医生胜出，总体却输 | 每份单独证据增加嫌疑，合在一起却减少嫌疑 |
>
> 两者的共同数学结构是：
>
> $$\frac{a_1}{b_1} > \frac{a_2}{b_2} \;\text{ 且 }\; \frac{c_1}{d_1} > \frac{c_2}{d_2} \quad\text{但}\quad \frac{a_1+c_1}{b_1+d_1} < \frac{a_2+c_2}{b_2+d_2}$$
>
> 这就是辛普森悖论的核心：**每个子组里的不等式方向，在合并子组后因为各组权重（分母 $b_i, d_i$）不同而发生反转。** 它在频率表里表现为"好医生总体成功率更低"，在贝叶斯更新里表现为"两份证据合在一起反而降低嫌疑"——同一个数学现象，穿了不同的衣服。

**(b)** 数学验证。这个证明的本质是：贝叶斯更新就是乘法，而乘法是结合的。

第一次更新（只看 $E_1$）：
$$P(H \mid E_1) = \frac{P(E_1 \mid H)P(H)}{P(E_1)}$$

第二次更新（在 $P(\cdot \mid E_1)$ 的基础上再看 $E_2$）：
$$P(H \mid E_1, E_2) = \frac{P(E_2 \mid H, E_1)P(H \mid E_1)}{P(E_2 \mid E_1)}$$

代入第一次的结果：
$$= \frac{P(E_2 \mid H, E_1) \cdot P(E_1 \mid H)P(H)}{P(E_2 \mid E_1) \cdot P(E_1)} = \frac{P(E_1, E_2 \mid H)P(H)}{P(E_1, E_2)}$$

最后一步用了链式法则：$P(E_2 \mid H, E_1)P(E_1 \mid H) = P(E_1, E_2 \mid H)$，以及 $P(E_2 \mid E_1)P(E_1) = P(E_1, E_2)$。

这就是直接同时更新。两者完全一致。

**直觉：** 贝叶斯更新是"可累积的"——信息到达的顺序不影响最终结论。你今天听到一条新闻更新了看法，明天又听到另一条新闻再更新一次，和两天后一次性知道两条新闻，最终的判断是一样的。这个性质是贝叶斯推断**一致性**的数学基础。

### 题 5：与未知对手下棋（条件独立的经典）⭐⭐

**来源：** Stat 110 战略练习 2 (SP2)，3.5

> 你的对手等可能是一个初学者（你赢的概率 90%）、中级（50%）或高手（30%）。你下了两局。
> (a) 你赢第一局的概率是多少？
> (b) 如果你赢了第一局，赢第二局的概率是多少？
> (c) 讨论无条件独立与条件独立（给定对手实力下独立）的区别。

**▼ 答案与解析**

设 $W_1, W_2$ 表示赢第一局和第二局，$S \in \{\text{初}, \text{中}, \text{高}\}$ 表示对手实力，每种概率 1/3。

**(a)** 用 LOTP：

$$P(W_1) = \frac{1}{3}(0.9 + 0.5 + 0.3) = \frac{1}{3} \times 1.7 = \frac{17}{30}$$

**(b)** 要求 $P(W_2 \mid W_1)$。需要计算 $P(W_1 \cap W_2)$ 和 $P(W_1)$。

给定对手实力 $S$，$W_1$ 和 $W_2$ **条件独立**：
$$P(W_1 \cap W_2 \mid S) = P(W_1 \mid S) P(W_2 \mid S)$$

所以：
$$P(W_1 \cap W_2) = \frac{1}{3}(0.9^2 + 0.5^2 + 0.3^2) = \frac{1}{3}(0.81 + 0.25 + 0.09) = \frac{1.15}{3} = \frac{23}{60}$$

因此：
$$P(W_2 \mid W_1) = \frac{P(W_1 \cap W_2)}{P(W_1)} = \frac{23/60}{17/30} = \frac{23}{34}$$

**(c)** 关键洞察：

- $P(W_2 \mid W_1) = 23/34 \approx 0.676$，而 $P(W_2) = 17/30 \approx 0.567$。前者显著大于后者，说明 $W_1$ 和 $W_2$ **不是无条件独立的**。
- 原因：赢了第一局 → 推断对手可能比较弱 → 提高第二局的预期胜率。
- 但给定 $S$（已经知道对手实力），$W_1$ 和 $W_2$ **是条件独立的**——因为知道了对手的实力，第一局的结果不提供任何关于第二局的额外信息。

这就是条件独立性的核心思想，也是贝叶斯网络中编码变量之间"信息流"的基本单元。

### 题 6：Monty Hall 问题（条件概率的直觉训练）⭐

**来源：** 教材 Chapter 2

> 三扇门，一扇后面有车，两扇后面是山羊。你选一扇门。主持人（知道车在哪）打开另一扇有山羊的门。你应该换门还是坚持原来的选择？

**▼ 答案与解析**

设 $C$ 表示你最初选中的那扇门后有车，$M$ 表示主持人打开某扇有山羊的门。

$$P(C) = 1/3$$

关键：以车的位置为条件。

- 如果你最初选对了（概率 1/3）：主持人有两扇山羊门可选，打开任一扇。你换门会输。
- 如果你最初选错了（概率 2/3）：主持人只能打开唯一的另一扇山羊门（因为剩下的那扇有车）。你换门会赢。

所以：
$$P(\text{换门赢}) = P(C^c) = 2/3$$
$$P(\text{坚持赢}) = P(C) = 1/3$$

**答案是换门，赢的概率从 1/3 提升到 2/3。**

**直觉：** 主持人的行为不是随机的——他必须避开有车的那扇门。这个约束提供了信息。条件于主持人打开的门，你的后验概率改变了。

### 题 7：两个信封问题（条件期望与概率陷阱）⭐

**来源：** 教材 Chapter 2 习题

> 两个信封，一个里面是另一个的两倍金额。你随机选了一个信封，打开看到 $x$ 元。你可以换另一个信封。换的理由是：另一个信封要么是 $x/2$ 要么是 $2x$，期望是 $(x/2+2x)/2 = 1.25x > x$，所以应该换。但如果这个逻辑成立，不管你打开看到多少钱，你都会想换——那为什么不一开始就选另一个？找出推理中的漏洞。

**▼ 答案与解析**

这个推理的漏洞在于：**它假设无论你看到多少钱，另一个信封里是 $x/2$ 和 $2x$ 的概率各是 1/2。这个假设不可能对所有 $x$ 都成立。**

具体来说，如果你看到的金额是 $x$，那么条件于看到 $x$，另一个信封是 $x/2$ 的概率和是 $2x$ 的概率并不是各 1/2——这取决于信封金额的先验分布。

设信封金额为 $(A, 2A)$（$A$ 是较小金额）。你用贝叶斯定理条件于你看到的金额 $x$：

- 如果 $x = A$（你拿到了较小的），那么另一个是 $2A = 2x$。
- 如果 $x = 2A$（你拿到了较大的），那么另一个是 $A = x/2$。

$P(x=A)$ 和 $P(x=2A)$ 的先验可能相等（对称），所以无条件概率各 1/2。但**条件于你观察到 $x$ 后**，$A$ 可能取多个值——$x$ 本身也可以是 $A$ 或者是 $2A$，这取决于 $A$ 的可能值。因此后验概率取决于 $A$ 的分布。

结论：期望 $1.25x$ 的计算依赖于不成立的概率假设。正确的计算必须引入 $A$ 的（合适的）先验分布，或认识到"无条件"的论证不成立。

**更深层的教训：** 在概率论中，你不能在没有明确的先验假设下进行条件期许的计算。"总是换"的论证之所以看起来自洽，恰是因为它偷偷地在每一步用了不一致的概率假设。

### 题 8：独立事件的构造（概念深化）⭐

**来源：** Stat 110 战略练习 2 (SP2)，Section 2

> (a) 一个事件可以和它自己独立吗？如果可以，什么时候？
> (b) 如果 $A$ 和 $B$ 独立，证明 $A^c$ 和 $B^c$ 也独立。
> (c) 构造三个事件两两独立但不相互独立的例子。
> (d) 构造 $P(A \cap B \cap C) = P(A)P(B)P(C)$ 但 $A, B, C$ 并不相互独立的例子。

**▼ 答案与解析**

**(a)** 可以，当且仅当 $P(A) = P(A)^2$，即 $P(A) = 0$ 或 $P(A) = 1$。解释：一个概率为 0 或 1 的事件和任何事件（包括它自己）都独立——因为它"几乎永远不发生"或"几乎永远发生"，不携带任何信息。

**(b)** 
$$P(A^c \cap B^c) = P((A \cup B)^c) = 1 - P(A \cup B)$$
$$= 1 - (P(A) + P(B) - P(A \cap B))$$
$$= 1 - P(A) - P(B) + P(A)P(B)$$
$$= (1-P(A))(1-P(B)) = P(A^c)P(B^c)$$

**(c)** 见上文 4.3 节的两枚硬币例子。

**(d)** 构造：令 $P(A) = 0$（即 $A$ 是零概率事件）。

则对任意 $B, C$：
$$P(A \cap B \cap C) \le P(A) = 0 \;\Longrightarrow\; P(A \cap B \cap C) = 0 = P(A)P(B)P(C)$$

乘积等式 $P(A \cap B \cap C) = P(A)P(B)P(C)$ 自动成立。

但是，令 $B = C$ 且 $0 < P(B) < 1$，则：
$$P(B \cap C) = P(B) \neq P(B)^2 = P(B)P(C)$$

所以 $B$ 和 $C$ 不独立，因此 $A, B, C$ 不相互独立（相互独立要求所有事件对独立）。

**关键洞察：** 零概率事件像一个"免费通行证"——它满足乘积等式，但不提供真正的独立性约束。这说明乘积等式只是相互独立的必要条件而非充分条件。

## 附录：充要条件的推导（5.3.2 节注释）

给定 $A \perp B \mid C$，即：

$$P(A \cap B \mid C) = P(A \mid C)P(B \mid C)$$
$$P(A \cap B \mid C^c) = P(A \mid C^c)P(B \mid C^c)$$

要求 $A \perp B$，即 $P(A \cap B) = P(A)P(B)$。

用 LOTP 展开左边：

$$P(A \cap B) = P(A \cap B \mid C)P(C) + P(A \cap B \mid C^c)P(C^c)$$
$$= P(A \mid C)P(B \mid C)P(C) + P(A \mid C^c)P(B \mid C^c)P(C^c) \tag{1}$$

用 LOTP 展开右边：

$$P(A)P(B) = [P(A \mid C)P(C) + P(A \mid C^c)P(C^c)] \times [P(B \mid C)P(C) + P(B \mid C^c)P(C^c)]$$
$$= P(A \mid C)P(B \mid C)P(C)^2 + P(A \mid C^c)P(B \mid C^c)P(C^c)^2$$
$$\quad + [P(A \mid C)P(B \mid C^c) + P(A \mid C^c)P(B \mid C)]P(C)P(C^c) \tag{2}$$

令 (1) = (2)，移项整理：

$$P(A \mid C)P(B \mid C)[P(C) - P(C)^2] + P(A \mid C^c)P(B \mid C^c)[P(C^c) - P(C^c)^2]$$
$$= [P(A \mid C)P(B \mid C^c) + P(A \mid C^c)P(B \mid C)]P(C)P(C^c)$$

注意 $P(C) - P(C)^2 = P(C)(1-P(C)) = P(C)P(C^c)$，同理 $P(C^c) - P(C^c)^2 = P(C^c)P(C)$。代入：

$$P(C)P(C^c)[P(A \mid C)P(B \mid C) + P(A \mid C^c)P(B \mid C^c)]$$
$$= P(C)P(C^c)[P(A \mid C)P(B \mid C^c) + P(A \mid C^c)P(B \mid C)]$$

假设 $0 < P(C) < 1$（否则 $C$ 退化为常值，条件独立 = 无条件独立），除以 $P(C)P(C^c) \neq 0$：

$$P(A \mid C)P(B \mid C) + P(A \mid C^c)P(B \mid C^c) = P(A \mid C)P(B \mid C^c) + P(A \mid C^c)P(B \mid C)$$

移项并因式分解：

$$P(A \mid C)[P(B \mid C) - P(B \mid C^c)] = P(A \mid C^c)[P(B \mid C) - P(B \mid C^c)]$$

$$[P(A \mid C) - P(A \mid C^c)][P(B \mid C) - P(B \mid C^c)] = 0$$

这就完成了推导。条件等价于 $P(A \mid C) = P(A \mid C^c)$ 或 $P(B \mid C) = P(B \mid C^c)$，即 $A \perp C$ 或 $B \perp C$。
