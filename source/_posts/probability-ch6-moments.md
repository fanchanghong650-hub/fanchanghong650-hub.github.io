## 为什么第六章是"从数字到形状"的升华

前五章你已经掌握了概率论的核心语言：概率、条件、随机变量、期望、连续分布。第四章告诉你期望是"平均而言会发生什么"，第五章让你能处理连续量。

但有一个问题悬而未决：**如何完整地刻画一个分布？**

均值和方差只是两个数字——它们能区分不同分布吗？不能。两个完全不同的分布可以有相同的均值和方差。你需要更多的"描述符"。

第六章给出的答案是**矩 (Moments)**。$n$ 阶矩 $E(X^n)$ 就像分布的 DNA 序列：如果两个分布的**所有**矩都相同，它们就是同一个分布。而**矩母函数 (Moment Generating Function, MGF)** 一次性打包了所有的矩，让你可以用一个函数完整描述整个分布。

这一章的三条主线：
1. **低阶矩的几何意义**：方差告诉你"有多散"，协方差告诉你"两个变量怎么一起变"，偏度和峰度告诉你分布的"形状"
2. **矩母函数作为统一框架**：MGF 把求矩变成求导，把独立和变成乘积，把一个分布的特征浓缩进一个函数
3. **矩与独立性的深度关系**：独立 $\Rightarrow$ 不相关（协方差为零），但反过来不成立——MGF 可以给出更强的独立性刻画

## 一、方差与标准差：从"中心"到"散布"

### 1.1 方差的定义与两种计算方式

第四章定义期望为"平均位置"，而方差告诉你"离平均有多远"：

$$E(X) = \mu, \quad \text{Var}(X) = E[(X - \mu)^2]$$

**定义**：方差是 $(X - \mu)^2$ 的期望——偏离均值的平方的平均值。标准差 $\text{SD}(X) = \sqrt{\text{Var}(X)}$ 将其还原到原始单位。

**两种计算方式**：

1. 定义式：$\text{Var}(X) = E[(X - EX)^2]$
2. 实用式：$\text{Var}(X) = E(X^2) - (EX)^2$

实用式几乎总是更容易计算。证明只需要展开平方：

$$\text{Var}(X) = E(X^2 - 2XEX + (EX)^2) = E(X^2) - 2(EX)^2 + (EX)^2 = E(X^2) - (EX)^2$$

第二行用到了 $E(2XEX) = 2EX \cdot EX = 2(EX)^2$（注意 $EX$ 是常数，可以提出）。

> **为什么平方而不是绝对值？**
>
> 用绝对离差 $E|X - \mu|$ 直觉上更自然。但平方有两个压倒性优势：(1) 平方是可微的，绝对值在求导时会产生麻烦；(2) 平方配合期望的线性性质产生了一个优美的恒等式 $\text{Var}(X) = E(X^2) - (EX)^2$。绝对值没有这样的代数结构。数学中选择平方不是因为"自然"，而是因为"管用"。

### 1.2 方差的核心性质

设 $X, Y$ 为随机变量，$c$ 为常数：

1. **常数方差为零**：$\text{Var}(c) = 0$——常数不波动
2. **常数平方提出**：$\text{Var}(cX) = c^2 \text{Var}(X)$——方差对尺度是二次的
3. **和的方差（关键区别）**：

$$\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)$$

只有当 $X$ 和 $Y$ **不相关**时，才有 $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y)$。

> **方差 vs 期望的关键差异**：期望的线性性质不需要任何条件——$E(X+Y)$ 永远等于 $E(X) + E(Y)$。但方差有这个额外的协方差项。这是为什么我们需要下一节的内容。

### 1.3 常见分布的方差

| 分布 | 期望 | 方差 | 直觉 |
|------|------|------|------|
| $\text{Bern}(p)$ | $p$ | $p(1-p)$ | $p=0.5$ 时方差最大（最不确定） |
| $\text{Bin}(n,p)$ | $np$ | $np(1-p)$ | 独立 Bernoulli 之和 |
| $\text{Geom}(p)$ | $1/p$ | $(1-p)/p^2$ | 等待时间的不确定性 |
| $\text{Pois}(\lambda)$ | $\lambda$ | $\lambda$ | 泊松的均值=方差，这是其特征 |
| $\text{Unif}(a,b)$ | $(a+b)/2$ | $(b-a)^2/12$ | 范围越宽方差越大 |
| $\text{Expo}(\lambda)$ | $1/\lambda$ | $1/\lambda^2$ | 标准差=均值，无记忆性的代价 |
| $\text{N}(\mu,\sigma^2)$ | $\mu$ | $\sigma^2$ | 方差直接作为参数出现 |

注意 Poisson 的均值=方差是一个**诊断特征**：如果数据远偏离这个关系，Poisson 分布就不是好模型。

## 二、协方差与相关系数：量化"一起变"的程度

### 2.1 协方差的定义与直觉

**定义**：$X$ 和 $Y$ 的协方差为

$$\text{Cov}(X, Y) = E[(X - EX)(Y - EY)]$$

等价的计算形式：$\text{Cov}(X, Y) = E(XY) - E(X)E(Y)$。

**直觉**：把平面按 $(EX, EY)$ 分成四个象限。如果 $X$ 和 $Y$ "同向变化"——当 $X$ 大于均值时 $Y$ 也倾向于大于均值——那么乘积 $(X-EX)(Y-EY)$ 趋向正值，协方差为正。反之，如果它们"反向变化"，协方差为负。如果协方差为零，意味着 $X$ 和 $Y$ **不相关**（没有线性关系）。

> **协方差的几何解释**：如果把 $X - EX$ 和 $Y - EY$ 看作向量，协方差就是它们的内积，相关系数就是它们夹角的余弦。方差是向量与自身的内积。

### 2.2 协方差的双线性性质

协方差最重要的代数性质是**双线性**——对每个参数都是线性的：

$$\text{Cov}(aX + bY, Z) = a\text{Cov}(X, Z) + b\text{Cov}(Y, Z)$$
$$\text{Cov}(X, cY + dZ) = c\text{Cov}(X, Y) + d\text{Cov}(X, Z)$$

这让你可以把复杂的协方差拆解成基本组件的协方差。例如：

$$\text{Var}(X+Y) = \text{Cov}(X+Y, X+Y) = \text{Cov}(X,X) + \text{Cov}(X,Y) + \text{Cov}(Y,X) + \text{Cov}(Y,Y)$$
$$= \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X,Y)$$

同样的逻辑可以推广到 $n$ 个变量：

$$\text{Var}\left(\sum_{i=1}^n X_i\right) = \sum_{i=1}^n \text{Var}(X_i) + 2\sum_{i < j} \text{Cov}(X_i, X_j)$$

> **理解协方差双线性的一个类比**：它和乘法对加法的分配律 $(a+b)(c+d) = ac + ad + bc + bd$ 同构。协方差就是随机变量空间中的"乘法"。

### 2.3 相关系数：去量纲的标准化度量

协方差的问题是它的量纲：如果 $X$ 的单位是米，$Y$ 的单位是秒，协方差的单位是米×秒——难以直观理解。相关系数解决了这个问题：

$$\text{Corr}(X, Y) = \rho = \frac{\text{Cov}(X, Y)}{\text{SD}(X) \cdot \text{SD}(Y)}$$

**性质**：
- $-1 \leq \rho \leq 1$
- $\rho = 1$ $\Leftrightarrow$ $Y = aX + b$ 且 $a > 0$（完全正线性）
- $\rho = -1$ $\Leftrightarrow$ $Y = aX + b$ 且 $a < 0$（完全负线性）
- $\rho = 0$ $\Leftrightarrow$ $\text{Cov}(X,Y) = 0$（无线性关系）

**为什么 $-1 \leq \rho \leq 1$？** 这来自 Cauchy-Schwarz 不等式：$|\text{Cov}(X,Y)| \leq \text{SD}(X) \cdot \text{SD}(Y)$。如果你把 $X - EX$ 和 $Y - EY$ 看作两个向量，相关系数就是它们夹角的余弦——自然在 $[-1, 1]$ 之间。

> **最关键的一点：不相关 ≠ 独立**
>
> 独立 $\Rightarrow$ 协方差为零 $\Rightarrow$ 不相关。但反过来不成立。一个经典反例：$X \sim N(0,1)$，$Y = X^2$。因为正态分布关于 0 对称，$E(XY) = E(X^3) = 0 = E(X) \cdot 0 = 0$，所以 $\text{Cov}(X,Y) = 0$。但显然 $X$ 和 $X^2$ 不独立——知道 $X$ 就完全确定了 $Y$。协方差只能捕捉**线性**关系。

## 三、偏度与峰度：分布形状的高阶描述

### 3.1 为什么前两阶矩不够

均值和方差分别刻画了**位置**和**散布**。但两个分布可以有相同的均值和方差，形状却完全不同：

- 一个可能左偏（长尾在左边），另一个对称
- 一个可能有厚尾（极端值频繁出现），另一个尾巴很薄

高阶矩填补了这些信息缺口。**$n$ 阶矩**定义为 $\mu_n = E(X^n)$，而**$n$ 阶中心矩**为 $E[(X - \mu)^n]$。

### 3.2 偏度：对称性的度量

**定义**：标准化三阶中心矩

$$\text{Skew}(X) = E\left[\left(\frac{X - \mu}{\sigma}\right)^3\right]$$

**直觉**：
- **正偏（右偏）**：Skew $> 0$，右尾比左尾长。例如指数分布 $\text{Skew} = 2$，以及大多数财富分布——少数人拥有极高财富拉长了右尾。
- **负偏（左偏）**：Skew $< 0$，左尾比右尾长。例如退休年龄——绝大多数人在 60-70 岁退休，但少数人提前退休拉长了左尾。
- **对称分布**：Skew $= 0$（但注意，Skew=0 不一定意味着对称）。

举例：$\text{Expo}(\lambda)$ 的偏度为 2（右偏），$\text{N}(\mu,\sigma^2)$ 的偏度为 0（对称），$\text{Bern}(p)$ 的偏度在 $p=0.5$ 时为 0，$p$ 偏离 0.5 时不为零。

### 3.3 峰度：尾部分量的度量

**定义**：标准化四阶中心矩

$$\text{Kurt}(X) = E\left[\left(\frac{X - \mu}{\sigma}\right)^4\right] - 3$$

减去 3 是为了让正态分布的峰度 = 0（称为"超额峰度"）。

**直觉**：峰度测量的是**尾部的厚度**。四阶矩 $E[(X-\mu)^4]$ 对偏离均值很远的值赋予极大的权重（四次方！），因此它主要受极端值的影响。

- **高峰度（厚尾）**：Kurt $> 0$，极端值比正态分布更常见。金融收益率是典型例子——崩盘发生的频率远超正态分布预测。
- **正态峰度**：Kurt $= 0$（或严格地说，原始峰度 = 3）。
- **低峰度（薄尾）**：Kurt $< 0$，极端值比正态分布更少见。例如均匀分布。

> **实践中为什么峰度很重要？** 金融风险管理中，如果假设收益率是正态分布而实际是厚尾分布，你会严重低估极端亏损的概率。2008 年金融危机部分原因就是模型低估了尾部风险。

## 四、矩母函数：统一矩的生成器

### 4.1 MGF 的定义与核心直觉

如果矩 $E(X), E(X^2), E(X^3), \ldots$ 是分布的"切片"，那么矩母函数就是把所有切片一次性装进一个函数的"压缩包"。

**定义**：随机变量 $X$ 的矩母函数为

$$M(t) = E(e^{tX})$$

其中 $t$ 是实数，且该期望在 $t=0$ 的某个邻域内存在。

**为什么叫"矩母函数"？** 将 $M(t)$ 在 $t=0$ 处 Taylor 展开：

$$M(t) = E\left(1 + tX + \frac{t^2 X^2}{2!} + \frac{t^3 X^3}{3!} + \cdots\right) = 1 + \frac{E(X)}{1!}t + \frac{E(X^2)}{2!}t^2 + \frac{E(X^3)}{3!}t^3 + \cdots$$

$t^n/n!$ 的系数恰好是 $E(X^n)$。因此：

$$E(X^n) = M^{(n)}(0)$$

在 $t=0$ 处求 $n$ 阶导数，就得到 $n$ 阶矩。

> **Taylor 级数只是理解 MGF 的直觉**——它不一定收敛。MGF 需要的是 $E(e^{tX})$ 在 $t=0$ 邻域内有限，这等价于 $X$ 的尾部分布不能太重。Cauchy 分布就没有 MGF——它的尾太重了。

### 4.2 MGF 的两大用途

**用途一：计算矩。** 对 MGF 求导通常比直接积分 $E(X^n)$ 容易得多。

**例子**：$X \sim \text{Expo}(\lambda)$，$M(t) = \frac{\lambda}{\lambda - t} = (1 - t/\lambda)^{-1}$（$t < \lambda$）。

利用几何级数展开：$M(t) = \sum_{n=0}^\infty \left(\frac{t}{\lambda}\right)^n = \sum_{n=0}^\infty \frac{n!}{\lambda^n} \cdot \frac{t^n}{n!}$

读出 $E(X^n) = n! / \lambda^n$——一次得到了**所有**矩。

**用途二：识别分布。** MGF 的一个重要定理：**如果两个随机变量在 $t=0$ 的邻域内有相同的 MGF，则它们有相同的分布。** 这让你可以通过比较 MGF 来识别一个随机变量服从什么分布。

### 4.3 MGF 的运算性质

MGF 把复杂的随机变量运算转化为简单的函数运算：

1. **线性变换**：如果 $Y = a + bX$，则 $M_Y(t) = e^{at} M_X(bt)$
2. **独立和**：如果 $X_1, \ldots, X_n$ 独立，则 $M_{X_1+\cdots+X_n}(t) = M_{X_1}(t) \cdots M_{X_n}(t)$
3. **独立和的逆**：如果独立和的 MGF 等于某个已知分布的 MGF，则和服从该分布

第三条是最强大的：它让你可以**不通过卷积**直接确定独立和的分布。例如，$n$ 个独立的 $\text{Expo}(\lambda)$ 之和的 MGF 为 $(1 - t/\lambda)^{-n}$，这恰好是 $\text{Gamma}(n, \lambda)$ 的 MGF。

### 4.4 常见分布的 MGF 速查表

| 分布 | MGF $M(t)$ | 定义域 |
|------|-----------|--------|
| $\text{Bern}(p)$ | $1-p + pe^t$ | 所有 $t$ |
| $\text{Bin}(n,p)$ | $(1-p + pe^t)^n$ | 所有 $t$ |
| $\text{Pois}(\lambda)$ | $e^{\lambda(e^t - 1)}$ | 所有 $t$ |
| $\text{Expo}(\lambda)$ | $\frac{\lambda}{\lambda - t} = (1 - t/\lambda)^{-1}$ | $t < \lambda$ |
| $\text{N}(0,1)$ | $e^{t^2/2}$ | 所有 $t$ |
| $\text{N}(\mu,\sigma^2)$ | $e^{\mu t + \sigma^2 t^2 / 2}$ | 所有 $t$ |
| $\text{Unif}(0,1)$ | $(e^t - 1)/t$ | 所有 $t \neq 0$ |
| $\text{Gamma}(n,\lambda)$ | $(1 - t/\lambda)^{-n}$ | $t < \lambda$ |

注意 MGF 的乘法结构：$\text{Bin}(n,p)$ 的 MGF 是 $\text{Bern}(p)$ MGF 的 $n$ 次方——对应 $n$ 个独立 Bernoulli 的和。$\text{Gamma}(n,\lambda)$ 的 MGF 是 $\text{Expo}(\lambda)$ MGF 的 $n$ 次方——对应 $n$ 个独立 Exponential 的和。这不是巧合，这是 MGF 设计的本意。

## 六、精选习题

### 第一组：方差、协方差与相关性（题 1-3）

**题 1（SP8-1 | 两骰子的协方差）** 掷两枚公平的六面骰子，绿色骰子结果为 $X$，橙色骰子结果为 $Y$。

(a) 求 $\text{Cov}(X+Y, X-Y)$。

(b) $X+Y$ 和 $X-Y$ 是否独立？为什么？

**▼ 答案与解析**

(a) 利用协方差的双线性展开：

$$\text{Cov}(X+Y, X-Y) = \text{Cov}(X,X) - \text{Cov}(X,Y) + \text{Cov}(Y,X) - \text{Cov}(Y,Y)$$
$$= \text{Var}(X) - \text{Cov}(X,Y) + \text{Cov}(Y,X) - \text{Var}(Y)$$

两枚骰子是独立的，所以 $\text{Cov}(X,Y) = 0$。又因为两骰子同分布，$\text{Var}(X) = \text{Var}(Y) = \frac{35}{12}$。因此：

$$\text{Cov}(X+Y, X-Y) = \frac{35}{12} - 0 + 0 - \frac{35}{12} = 0$$

(b) **不独立。** 协方差为零不等于独立。例如，如果知道 $X+Y = 12$，则必然 $X=6, Y=6$，所以 $X-Y$ 只能等于 0——知道了 $X+Y$ 就约束了 $X-Y$ 的取值。协方差只能检测**线性**关系，而 $X+Y$ 和 $X-Y$ 之间有明确的关系（它们实际上是 $X$ 和 $Y$ 的线性组合），只是这个关系的协方差恰好为零。

**洞察**：这是协方差为零但变量不独立的教科书级例子。$X+Y$ 和 $X-Y$ 似乎是相关的——但如果方差相同，它们的协方差恰好抵消。

**训练点**：协方差双线性展开、独立性与不相关的区别。

**题 2（SP8-4 | 多项分布的边际协方差）** 设 $(X_1, \ldots, X_k) \sim \text{Mult}(n, p_1, \ldots, p_k)$。证明对于 $i \neq j$：

$$\text{Cov}(X_i, X_j) = -n p_i p_j$$

然后解释为什么协方差为**负**。

**▼ 答案与解析**

将 $X_i$ 和 $X_j$ 分别写为 $n$ 次独立试验的指示变量之和：

$$X_i = \sum_{r=1}^n I_{i}^{(r)}, \quad X_j = \sum_{r=1}^n I_{j}^{(r)}$$

其中 $I_i^{(r)} = 1$ 如果第 $r$ 次试验结果为类别 $i$。由协方差的双线性：

$$\text{Cov}(X_i, X_j) = \sum_{r=1}^n \sum_{s=1}^n \text{Cov}(I_i^{(r)}, I_j^{(s)})$$

不同试验之间独立，所以当 $r \neq s$ 时协方差为零。只需考虑 $r = s$：

$$\text{Cov}(I_i^{(r)}, I_j^{(r)}) = E(I_i^{(r)} I_j^{(r)}) - E(I_i^{(r)})E(I_j^{(r)}) = 0 - p_i p_j = -p_i p_j$$

一次试验不能同时属于两个类别，所以 $I_i I_j = 0$。求和得 $\text{Cov}(X_i, X_j) = n \cdot (-p_i p_j) = -n p_i p_j$。

**为什么为负？** 在多项分布中，如果类别 $i$ 出现得很多，就会"挤占"类别 $j$ 的空间——总数固定为 $n$。这是一种**零和竞争**关系。

**洞察**：多项分布中不同类别的计数总是负相关的——因为总和固定为 $n$。这和多项分布的本质密切相关。

**训练点**：指示变量分解、协方差双线性、固定总数下的负相关直觉。

**题 3（教材 Ch6 | 超几何的方差）** 坛中有 $w$ 个白球和 $b$ 个黑球，无放回地抽取 $n$ 个球。设 $X$ 为抽中白球的数量。

(a) 将 $X$ 写为指示变量之和，$X = I_1 + \cdots + I_n$，其中 $I_j$ 指示第 $j$ 次抽到白球。

(b) 求 $\text{Var}(X)$。（提示：利用协方差的对称性，注意每次抽取的边缘概率相同。）

**▼ 答案与解析**

(a) $I_j = 1$ 如果第 $j$ 次抽到白球。则 $X = \sum_{j=1}^n I_j$。

(b) 利用方差的和公式：

$$\text{Var}(X) = \sum_{j=1}^n \text{Var}(I_j) + 2\sum_{1 \leq i < j \leq n} \text{Cov}(I_i, I_j)$$

每个 $I_j$ 的边缘分布：$P(I_j = 1) = \frac{w}{w+b}$（无放回时每次抽取的边缘概率相同）。所以：

$$\text{Var}(I_j) = \frac{w}{w+b} \cdot \frac{b}{w+b}$$

对于 $i < j$：

$$\text{Cov}(I_i, I_j) = P(I_i=1, I_j=1) - P(I_i=1)P(I_j=1)$$

$$= \frac{w}{w+b} \cdot \frac{w-1}{w+b-1} - \left(\frac{w}{w+b}\right)^2$$

简化后得到负值（因为前面的抽取消耗了一个白球，降低了后面再次抽到白球的概率）。将所有项求和：

$$\text{Var}(X) = n\frac{wb}{(w+b)^2} \cdot \frac{w+b-n}{w+b-1}$$

最后的因子 $\frac{w+b-n}{w+b-1}$ 是**有限总体修正 (FPC)**。当 $n=w+b$（抽光）时，方差为零——你确定了所有的白球数。当 $w+b \to \infty$ 时，FPC $\to 1$，退化为二项分布 $\text{Bin}(n, \frac{w}{w+b})$ 的方差。

**洞察**：超几何分布和二项分布有相同的期望 $n\frac{w}{w+b}$，但方差差了一个 FPC 因子。无放回意味着负相关——早期抽到白球减少了后期再抽到的机会，因此方差更小。

**训练点**：指示变量 + 协方差求方差、有限总体修正、超几何 vs 二项的联系。

### 第二组：矩母函数基础（题 4-6）

**题 4（SP6-2.1 | MGF 求指数分布的矩）** 设 $X \sim \text{Expo}(\lambda)$，其 MGF 为 $M(t) = (1 - t/\lambda)^{-1}$（$t < \lambda$）。

(a) 将 $M(t)$ 展开为 $t$ 的幂级数，读出 $E(X^n)$ 对任意 $n$ 的表达式。

(b) 利用 (a) 求 $E(X^3)$。

**▼ 答案与解析**

(a) 由几何级数公式，对 $|t/\lambda| < 1$：

$$M(t) = \frac{1}{1 - t/\lambda} = \sum_{n=0}^\infty \left(\frac{t}{\lambda}\right)^n = \sum_{n=0}^\infty \frac{n!}{\lambda^n} \cdot \frac{t^n}{n!}$$

由于 $M(t) = \sum_{n=0}^\infty E(X^n) \frac{t^n}{n!}$，系数对应给出：

$$E(X^n) = \frac{n!}{\lambda^n}$$

(b) $E(X^3) = 3! / \lambda^3 = 6 / \lambda^3$。

验证：由第四章，$E(X) = 1/\lambda$，$\text{Var}(X) = 1/\lambda^2$，而 $E(X^2) = 2/\lambda^2$，确实与 $E(X^2) = \text{Var}(X) + (EX)^2 = 1/\lambda^2 + 1/\lambda^2 = 2/\lambda^2$ 一致。

**洞察**：MGF 展开法一次给出了**所有**矩——而不需要对每个 $n$ 重新积分。这就是 MGF 的威力：一个级数展开胜过一百次分部积分。

**训练点**：MGF 的级数展开、从 Taylor 级数中读取矩、几何级数。

**题 5（SP6-2.2 | MGF 的线性变换）** 设随机变量 $X$ 的 MGF 为 $M_X(t)$。

(a) 求 $-X$ 的 MGF。

(b) 求 $a + bX$ 的 MGF（$a, b$ 为常数）。

**▼ 答案与解析**

(a) $M_{-X}(t) = E(e^{t(-X)}) = E(e^{(-t)X}) = M_X(-t)$。

(b) $M_{a+bX}(t) = E(e^{t(a+bX)}) = E(e^{at} \cdot e^{btX}) = e^{at} E(e^{(bt)X}) = e^{at} M_X(bt)$。

**洞察**：这两个变换规则是 MGF 的"语法"。掌握它们之后，任何线性变换的 MGF 都可以机械地写出。特别地，$e^{at}$ 只影响"位置"（加性平移），$M_X(bt)$ 中的 $bt$ 影响"尺度"。

**训练点**：MGF 的定义、期望中提出常数、函数变换。

**题 6（SP6-2.3 | 独立均匀随机变量和的 MGF）** 设 $U_1, \ldots, U_{60}$ 独立同分布于 $\text{Unif}(0,1)$，$X = U_1 + \cdots + U_{60}$。求 $X$ 的 MGF。

**▼ 答案与解析**

单个 $\text{Unif}(0,1)$ 的 MGF：

$$M_U(t) = E(e^{tU}) = \int_0^1 e^{tu} du = \frac{e^t - 1}{t} \quad (t \neq 0)$$

$t = 0$ 时 $M_U(0) = 1$。

由于 60 个 $U_i$ 独立，$X$ 的 MGF 为乘积：

$$M_X(t) = [M_U(t)]^{60} = \left(\frac{e^t - 1}{t}\right)^{60} \quad (t \neq 0)$$

$t = 0$ 时 $M_X(0) = 1$。

由中心极限定理，当 $n=60$ 时 $X$ 近似于 $N(30, 5)$。但 MGF 给出的是**精确**分布的信息——如果你想求 $E(X^k)$，直接对 $M_X(t)$ 求导即可，不需要近似。

**洞察**：独立和的 MGF 等于 MGF 的乘积——这是 MGF 最重要的性质。它把卷积（困难的积分）变成了乘法（简单的代数）。

**训练点**：独立和 MGF 的乘积法则、均匀分布的 MGF、MGF 在 $t=0$ 的连续性。

### 第三组：MGF 的性质与分布识别（题 7-8）

**题 7（SP6-2.4 | 泊松分布的累积量）** 设 $X \sim \text{Pois}(\lambda)$，MGF 为 $M(t) = e^{\lambda(e^t - 1)}$。定义累积量生成函数 $g(t) = \ln M(t)$，将其展开为 $g(t) = \sum_{j=1}^\infty c_j \frac{t^j}{j!}$。求所有 $j \geq 1$ 的 $c_j$（第 $j$ 个累积量）。

**▼ 答案与解析**

$$g(t) = \ln M(t) = \lambda(e^t - 1)$$

将 $e^t$ Taylor 展开：$e^t = \sum_{j=0}^\infty \frac{t^j}{j!}$。减去 1 后，$j=0$ 项消去：

$$g(t) = \lambda \sum_{j=1}^\infty \frac{t^j}{j!} = \sum_{j=1}^\infty \lambda \cdot \frac{t^j}{j!}$$

因此 $c_j = \lambda$ 对**所有** $j \geq 1$。

Poisson 的每个累积量都等于 $\lambda$。特别地：$c_1 = E(X) = \lambda$（均值），$c_2 = \text{Var}(X) = \lambda$（方差印证了均值=方差），$c_3 = \lambda$（偏度 = $\lambda/\lambda^{3/2} = 1/\sqrt{\lambda}$——样本越大，偏度越小），$c_4 = \lambda$。

**洞察**：累积量是矩的另一种"包装"。对于 Poisson，所有累积量都等于 $\lambda$——这是一个非常特殊的特征，意味着"知道 $\lambda$ 就知道了一切"。

**训练点**：MGF 与累积量生成函数的关系、Poisson 的 MGF、Taylor 展开。

**题 8（教材 Ch6 | MGF 识别独立正态和）** 设 $X_1 \sim N(\mu_1, \sigma_1^2)$，$X_2 \sim N(\mu_2, \sigma_2^2)$ 独立。

(a) 写出 $N(\mu_i, \sigma_i^2)$ 的 MGF。

(b) 利用 MGF 求 $X_1 + X_2$ 的分布。

(c) 推广到 $n$ 个独立正态随机变量的和。

**▼ 答案与解析**

(a) 对 $Z \sim N(0,1)$：$M_Z(t) = E(e^{tZ}) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty e^{tz - z^2/2} dz = e^{t^2/2}$。（配方后积分。）

对 $X_i = \mu_i + \sigma_i Z$：由题 5(b)，$M_{X_i}(t) = e^{\mu_i t} M_Z(\sigma_i t) = e^{\mu_i t} e^{\sigma_i^2 t^2 / 2} = e^{\mu_i t + \sigma_i^2 t^2 / 2}$。

(b) 由独立性，$M_{X_1+X_2}(t) = M_{X_1}(t) M_{X_2}(t) = e^{\mu_1 t + \sigma_1^2 t^2/2} \cdot e^{\mu_2 t + \sigma_2^2 t^2/2} = e^{(\mu_1+\mu_2)t + (\sigma_1^2 + \sigma_2^2)t^2/2}$。

这正是 $N(\mu_1+\mu_2, \sigma_1^2 + \sigma_2^2)$ 的 MGF。由于 MGF 唯一确定分布，$X_1 + X_2 \sim N(\mu_1+\mu_2, \sigma_1^2 + \sigma_2^2)$。

(c) 推广：$n$ 个独立正态随机变量之和 $\sim N(\sum \mu_i, \sum \sigma_i^2)$。

**洞察**：不用卷积！MGF 的乘积法则 + MGF 唯一确定分布 = 瞬间得到独立正态和的分布。这是 MGF 最经典的应用之一。

**训练点**：正态分布的 MGF 推导、MGF 乘积法则、MGF 分布识别、独立正态和的可加性。

## 七、本章核心脉络

本章可以用三个层次理解：

**第一层（低阶矩）**：方差量化散布，协方差和相关系数量化两个变量之间的**线性**关联强度。$\text{Var}(X+Y)$ 的公式告诉我们，方差不像期望那样对加法是"免费"的——它需要你考虑协方差。$\text{Cov} = 0$ 是独立性的必要条件，不是充分条件。

**第二层（高阶矩）**：偏度和峰度让你"看到"分布的形状——是对称的还是偏斜的？尾巴是厚的还是薄的？这四种矩（均值、方差、偏度、峰度）组合起来，能区分绝大多数常见分布。

**第三层（MGF）**：MGF 上升到另一个维度。它不只给出低阶矩——它一次性给出所有矩。更关键的是，它把"独立和"变成"MGF 乘积"，把"识别分布"变成"匹配 MGF"。这是整个概率论中最高级的操作。

**如果你只从这一章带走三样东西：**

1. **$\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X,Y)$**——永远记住协方差项。忘了它是在所有涉及方差相加的场景中犯下的最常见的错误。

2. **协方差为零 ≠ 独立**——这是概率论中最需要警惕的陷阱之一。协方差只测量线性关系。$X$ 和 $X^2$ 可以协方差为零但完全确定彼此。

3. **MGF 的三种操作**——求导得矩 → 乘积得独立和 → 匹配得分布。掌握了这三种操作，你就掌握了 MGF 的全部实用价值。
