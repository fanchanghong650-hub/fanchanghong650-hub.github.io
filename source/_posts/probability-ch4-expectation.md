## 为什么第四章是"从单变量到多变量思维"的转折点

第三章定义了随机变量和分布——你学会了用 PMF 和 CDF 描述一个随机变量的行为。但现实世界中的问题很少只涉及一个量：股票价格和利率、考试成绩和学习时间、基因型和表现型——你几乎总是在处理**多个**随机变量。

第四章的核心议题是：**当你面对多个随机变量时，如何描述它们之间的关系？**

这一章给出的答案是三个层次：
1. **期望的线性性质**——它不在乎变量之间有没有关系（最强大的免费午餐）
2. **协方差和相关系数**——它们精确量化变量之间的线性关系强度
3. **条件期望**——当你知道一些信息后，你的期望如何更新（把第二章的条件概率思想嫁接到期望上）

这三个层次层层递进：线性告诉你"不管怎样都可以直接加"，协方差告诉你"如果想知道能不能直接加方差，看这里"，条件期望告诉你"有了新信息后，一切重新算"。

## 一、期望的重新审视：从定义到线性性质

### 1.1 期望的基本性质

第三章我们知道了期望的定义：$E(X) = \sum_x x \cdot P(X=x)$。第四章首先系统地整理了期望的基本性质：

设 $X, Y$ 为随机变量，$c$ 为常数：

1. **常数的期望就是常数本身：** $E(c) = c$
2. **常数可以提出：** $E(cX) = cE(X)$
3. **可加性（最关键）：** $E(X+Y) = E(X) + E(Y)$
4. **保序性：** 如果 $P(X \geq Y) = 1$，则 $E(X) \geq E(Y)$。特别地，如果 $X \geq 0$ 几乎必然成立，则 $E(X) \geq 0$

其中第三条——**可加性**——是整个第四章乃至整本书的重中之重。

### 1.2 线性性质的完整陈述与证明

**定理（期望的线性性质）：** 对任意随机变量 $X_1, X_2, \ldots, X_n$ 和常数 $c_1, c_2, \ldots, c_n$：

$$E\left(\sum_{i=1}^n c_i X_i\right) = \sum_{i=1}^n c_i E(X_i)$$

**无论 $X_i$ 之间是否独立，这个等式永远成立。**

证明的核心在于期望的定义是对样本空间求和：

$$E(X+Y) = \sum_{\omega \in \Omega} (X(\omega) + Y(\omega)) \cdot P(\{\omega\}) = \sum_{\omega} X(\omega)P(\omega) + \sum_{\omega} Y(\omega)P(\omega) = E(X) + E(Y)$$

等式中没有任何一步需要 $X$ 和 $Y$ 独立的假设。期望的线性性质本质上是**求和在样本空间上的线性性质**——而求和本身永远是线性的。

> **为什么学生们经常低估这个定理？**
>
> 因为它太"简单"了——就是求和拆分。但它的威力不在于证明难度，而在于它让你可以**完全绕开联合分布**来计算期望。你不需要知道 $X$ 和 $Y$ 的联合 PMF，甚至不需要知道它们是否相关——只要你知道各自的边缘期望，你就可以把它们加起来。

### 1.3 线性期望与指示变量：概率论中最强大的组合技

指示变量法 + 线性期望 = 解决复杂期望问题的万能套路：

1. 把一个复杂的量 $X$ 分解为指示变量的和：$X = I_1 + I_2 + \cdots + I_n$
2. 对每个 $I_j$，$E(I_j) = P(I_j = 1)$（这个概率通常很容易算）
3. 用线性期望：$E(X) = \sum_{j=1}^n P(I_j = 1)$

不需要知道 $X$ 的完整分布。不需要知道 $I_j$ 之间的联合分布。不需要独立性假设。

**例：随机排列中的不动点数。** $n$ 张卡片随机排列，设 $X$ 为"恰好在自己原始位置上的卡片数"。求 $E(X)$。

设 $I_j$ 为第 $j$ 张卡片在原始位置的指示变量。由对称性，$P(I_j = 1) = 1/n$。于是 $E(X) = \sum_{j=1}^n E(I_j) = n \cdot 1/n = 1$。无论 $n$ 多大，平均而言恰好有 1 张卡片会回到原位。

如果不用指示变量法，你需要算出 $X$ 的完整 PMF（这涉及排列的容斥原理，非常繁琐）。指示变量法把复杂问题变成了几乎"免费"的计算。

## 二、方差：深化的理解

### 2.1 方差的性质回顾与补充

第三章给出了方差的定义和基本计算公式。第四章从"多变量关系"角度深化对方差的理解。

回顾核心公式：
- $\text{Var}(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2$
- $\text{Var}(cX) = c^2 \text{Var}(X)$
- $\text{Var}(X + c) = \text{Var}(X)$

### 2.2 两个随机变量和的方差

这是第四章最关键的方差公式：

$$\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)$$

其中 $\text{Cov}(X, Y)$ 是协方差（下一节详述）。这个公式可以推广到 $n$ 个变量的情形：

$$\text{Var}\left(\sum_{i=1}^n X_i\right) = \sum_{i=1}^n \text{Var}(X_i) + 2\sum_{i < j} \text{Cov}(X_i, X_j)$$

**关键对比：期望 vs 方差对独立性的依赖**

| 性质 | 期望 | 方差 |
|------|------|------|
| 和的拆分 | $E(X+Y) = E(X) + E(Y)$ **永远成立** | $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y)$ **仅当不相关时** |
| 线性组合 | $E(cX) = cE(X)$，线性 | $\text{Var}(cX) = c^2\text{Var}(X)$，二次 |
| 常数项 | $E(X+c) = E(X)+c$，影响期望 | $\text{Var}(X+c) = \text{Var}(X)$，不影响方差 |

直觉：方差衡量的是"离期望有多远"。加减一个常数把整个分布平移了，但离散程度不变，所以方差不变。乘以常数 $c$ 不仅平移而且拉伸，离散程度放大 $|c|$ 倍，方差放大 $c^2$ 倍。

### 2.3 独立 → 方差可加，但反向不成立

如果 $X$ 和 $Y$ 独立，则 $\text{Cov}(X, Y) = 0$，从而 $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y)$。

但协方差为 0 **不能推出**独立。一个经典的例子：

> $X \sim \text{Uniform}(-1, 0, 1)$（取 $-1, 0, 1$ 各以 $1/3$ 概率），定义 $Y = X^2$。
>
> 则 $E(X) = 0$，$E(XY) = E(X^3) = E(X) = 0$（因为 $X^3 = X$ 当 $X \in \{-1, 0, 1\}$），所以 $\text{Cov}(X, Y) = E(XY) - E(X)E(Y) = 0$。
>
> 但 $X$ 和 $Y$ 显然不独立——$Y$ 完全由 $X$ 决定（$Y = X^2$），知道 $X$ 就确定了 $Y$。

不相关 (uncorrelated) 只意味着没有**线性**关系。非线性关系（如平方关系）在协方差中是无法被检测的。

## 三、协方差与相关系数：量化关系的工具

### 3.1 协方差的定义与直觉

$$\text{Cov}(X, Y) = E\big[(X - E(X))(Y - E(Y))\big]$$

展开后得到更实用的计算形式：

$$\text{Cov}(X, Y) = E(XY) - E(X)E(Y)$$

**直觉：** 协方差衡量的是"$X$ 高于其期望时，$Y$ 是否也倾向于高于其期望"。

- 如果 $X$ 高于均值时 $Y$ 也倾向于高于均值，则 $(X-E(X))$ 和 $(Y-E(Y))$ 同号，乘积为正，协方差为正 → 正相关
- 如果 $X$ 高于均值时 $Y$ 倾向于低于均值，则两者异号，乘积为负，协方差为负 → 负相关
- 如果两者没什么系统性关系，正负互相抵消，协方差接近 0

**例：** 设 $X$ 表示每天学习的小时数，$Y$ 表示考试成绩。我们希望 $X$ 和 $Y$ 之间有正的协方差——学习时间越长，分数倾向于越高。

### 3.2 协方差的双线性性质

协方差最重要的代数性质是**双线性 (bilinearity)**：

$$\text{Cov}(aX + bY, Z) = a\text{Cov}(X, Z) + b\text{Cov}(Y, Z)$$

$$\text{Cov}(X, cY + dZ) = c\text{Cov}(X, Y) + d\text{Cov}(X, Z)$$

$$
\begin{aligned}
\text{Cov}(aX + bY, cU + dV) &= ac\text{Cov}(X, U) + ad\text{Cov}(X, V) \\
&+ bc\text{Cov}(Y, U) + bd\text{Cov}(Y, V)
\end{aligned}
$$

这个性质和期望的线性性质一脉相承：协方差在每一个参数中都是线性的。这使得方差的复杂组合可以通过逐项展开协方差来计算。

**例：利用双线性直接证明 $\text{Var}(X+Y)$ 公式：**

$$
\begin{aligned}
\text{Var}(X+Y) &= \text{Cov}(X+Y, X+Y) \\
&= \text{Cov}(X, X) + \text{Cov}(X, Y) + \text{Cov}(Y, X) + \text{Cov}(Y, Y) \\
&= \text{Var}(X) + 2\text{Cov}(X, Y) + \text{Var}(Y)
\end{aligned}
$$

注意这里用到了 $\text{Var}(X) = \text{Cov}(X, X)$——方差是一个随机变量与自身的协方差。

### 3.3 协方差的关键性质一览

| 性质 | 公式 |
|------|------|
| 对称性 | $\text{Cov}(X, Y) = \text{Cov}(Y, X)$ |
| 自身协方差 | $\text{Cov}(X, X) = \text{Var}(X)$ |
| 与常数 | $\text{Cov}(X, c) = 0$ |
| 双线性 | 见上节 |
| 独立 → 零协方差 | $X, Y$ 独立 $\implies$ $\text{Cov}(X, Y) = 0$ |
| 零协方差 ⇏ 独立 | 反例见 2.3 节 |

### 3.4 相关系数：去除量纲的标准化度量

协方差有一个实际问题：它的数值依赖于 $X$ 和 $Y$ 的单位。如果 $X$ 以厘米而非米为单位测量，协方差会放大 100 倍——但两个变量之间关系的**强度**并没有改变。

**相关系数 (correlation coefficient)** 解决了这个问题：

$$\text{Corr}(X, Y) = \frac{\text{Cov}(X, Y)}{\text{SD}(X) \cdot \text{SD}(Y)}$$

这是一个**无单位的量**（分子和分母的尺度相同），取值范围被限制在 $[-1, 1]$：

$$-1 \leq \text{Corr}(X, Y) \leq 1$$

**直觉：**
- $\text{Corr} = 1$：完美正线性关系（$Y = a + bX$ 其中 $b > 0$）
- $\text{Corr} = -1$：完美负线性关系（$Y = a + bX$ 其中 $b < 0$）
- $\text{Corr} = 0$：无线性关系（不相关）
- 中间值：有一定线性关系但非完美

**相关系数只衡量线性关系。** $Y = X^2$ 可以有接近 1 的相关性，也可以有接近 0 的相关性——取决于 $X$ 的分布。你在使用相关系数时必须记住这个局限：它告诉你"这两个变量有多接近一条直线"，而不是"这两个变量有没有关系"。

### 3.5 从协方差理解方差的可加性

现在可以更深刻地理解为什么独立/不相关是方差可加的前提：

$$\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)$$

- 当 $\text{Cov}(X, Y) = 0$（不相关）时，方差可加
- 当 $\text{Cov}(X, Y) > 0$（正相关）时，$\text{Var}(X+Y) > \text{Var}(X) + \text{Var}(Y)$——和比"各部分单独"更不稳定，因为两个变量倾向于同方向波动
- 当 $\text{Cov}(X, Y) < 0$（负相关）时，$\text{Var}(X+Y) < \text{Var}(X) + \text{Var}(Y)$——和比"各部分单独"更稳定，因为波动倾向于互相抵消

这就是为什么**分散投资降低风险**的数学基础——如果你持有的不同股票之间是负相关或弱相关的，组合的总体波动会比单只股票的波动小。

## 四、条件期望：把"以什么为条件"嫁接到期望上

### 4.1 条件于事件的期望

给定事件 $A$（其中 $P(A) > 0$），$X$ 在条件 $A$ 下的期望为：

$$E(X \mid A) = \sum_x x \cdot P(X = x \mid A)$$

这就是把普通的 PMF $P(X=x)$ 替换为条件 PMF $P(X=x \mid A)$，然后按相同的公式计算期望。

条件期望 $E(X \mid A)$ 是一个**数**——它是你在知道 $A$ 发生后，对 $X$ 的"最佳猜测"。

**例：** $X$ 为抛一枚公平硬币正面的次数（$X \sim \text{Bern}(1/2)$）。无条件期望：$E(X) = 0.5$。但如果你知道"硬币是双面都是正面的特殊硬币"（事件 $A$），则 $E(X \mid A) = 1$。新信息改变了你的期望。

### 4.2 全期望定律 (Law of Total Expectation / Adam's Law)

这是第四章最重要的定理之一——它是第二章的全概率公式在期望上的类比：

**定理：** 设 $B_1, B_2, \ldots, B_n$ 是样本空间的一个划分（互斥且穷尽），$P(B_j) > 0$ 对所有 $j$，则

$$E(X) = \sum_{j=1}^n E(X \mid B_j) \cdot P(B_j)$$

**直觉：** 你把所有可能的情况 $B_j$ 枚举出来，在每个情况下计算"条件于 $B_j$ 的期望"，然后用 $B_j$ 发生的概率做加权平均，就得到了无条件的总期望。

这个公式的威力在于：有些情况下直接求 $E(X)$ 很困难，但如果你对某个合适的划分进行条件化，$E(X \mid B_j)$ 就变得很简单。

> **为什么叫 Adam's Law？** 在 Stat 110 的传统中，全期望定律被称为"Adam's Law"——一个助记：全书第一作者叫 Joseph **A**dam Blitzstein 的中间名是 Adam。类似地，全方差定律被称为"Eve's Law"（但这个在后面的章节）。

### 4.3 Adam's Law 的经典应用：赌徒破产问题中的期望步数

一个赌徒每次以概率 $p$ 赢 1 元，以概率 $q = 1-p$ 输 1 元。他从 $i$ 元开始，目标是达到 $N$ 元（$0 < i < N$），达到 0 元则破产。设 $X$ 为他最终达到 0 或 $N$ 所需的步数。这个问题的期望用 Adam's Law 可以非常漂亮地解决。

条件于第一步的结果：

$$E(X \mid \text{初始资产 } = i) = 1 + p \cdot E(X \mid \text{初始资产 } = i+1) + q \cdot E(X \mid \text{初始资产 } = i-1)$$

这给出一组递推方程，然后可以用解差分方程的方法求闭式解。不需要模拟，不需要复杂的组合计数——Adam's Law 一步到位地把动态过程分解为"第一步 + 剩下的过程"。

这就是 Adam's Law 最典型的用法：**在序贯过程或分阶段问题中，条件于第一步（或第一阶段），把问题分解为"这一步的贡献 + 从新状态开始的期望"。**

### 4.4 Adam's Law 的另一个应用：随机个数随机变量的和的期望

假设你在一家商店，每天来的顾客数 $N$ 是随机的（$N \sim \text{Pois}(100)$），每位顾客的消费金额 $X_j \overset{i.i.d.}{\sim} \text{某分布}$，均值为 $E(X)$。那么每天总收入 $S = \sum_{j=1}^N X_j$ 的期望是多少？

$$E(S) = E\left(\sum_{j=1}^N X_j\right)$$

如果 $N$ 是随机的——这怎么算？用 Adam's Law，条件于 $N$：

$$E(S) = \sum_{n=0}^\infty E(S \mid N = n) \cdot P(N = n)$$

当 $N = n$ 固定时，$E(S \mid N = n) = \sum_{j=1}^n E(X_j) = n E(X)$。所以：

$$E(S) = \sum_{n=0}^\infty n E(X) \cdot P(N = n) = E(X) \cdot E(N)$$

**关键结果：随机个 i.i.d. 随机变量的和的期望 = 单项期望 × 个数的期望。** 但注意这要求 $X_j$ 之间独立且与 $N$ 也独立。

## 五、本章概念的全景图

第四章的核心概念之间的关系可以用下面的结构来理解：

```text
       期望 E(X)                          方差 Var(X)
          │                                  │
          ├─ 线性性质（不需要独立性）          ├─ Var(X+Y) = Var(X)+Var(Y)+2Cov(X,Y)
          │                                  │
          ├─ LOTUS: E(g(X)) = Σ g(x) P(X=x) ├─ 独立 ⇒ Cov=0 ⇒ 方差可加
          │                                  │
          ├─ 指示变量法                      ├─ 标准差 SD(X) = √Var(X)
          │   E(I_A) = P(A)                  │
          │                                  │
          └─ 亚当定律 (Adam's Law)           │
              E(X) = Σ E(X|Bⱼ) P(Bⱼ)        │
                                            │
                    ┌───────────────────────┘
                    │
                    ▼
              协方差 Cov(X,Y) = E(XY) - E(X)E(Y)
                    │
                    ├─ 双线性性质
                    │
                    └─ 相关系数 Corr(X,Y) = Cov(X,Y)/(SD(X)·SD(Y))
                         │
                         ├─ -1 ≤ Corr ≤ 1
                         ├─ Corr = ±1 ↔ 完美线性关系
                         └─ Corr = 0 ⇔ 无线性关系（但可能有非线性关系）
```

**最重要的层次关系：**

1. **无条件期望**是基础——你"不知道任何额外信息"时的最佳猜测
2. **条件期望**告诉你有了信息后如何更新猜测
3. **Adam's Law**把条件期望串联起来：无条件期望 = 条件期望的加权平均
4. **协方差**量化了两个变量围绕各自期望波动的"同步性"
5. **相关系数**是协方差的标准化版本，将关系强度压缩到 $[-1, 1]$

### 第一组：指示变量与线性期望（SP4 核心主题）

---

**题 1（SP4-2.3 | 鞋带问题——经典面试题）**

有 100 根鞋带（共 200 个自由端）。每一步随机选择两个自由端系在一起（所有自由端等可能被选中）。

(a) 直到所有自由端都系成环，需要多少步？
(b) 最终形成的环的期望数量是多少？

**▼ 答案与解析**

(a) 每步消耗 2 个自由端，从 200 个自由端到 0 个，恰好需要 100 步。无论你如何选择，步数是确定的——这不是随机变量。

(b) 设 $I_j$ 为第 $j$ 步形成闭环的指示变量（这里"第 $j$ 步"定义为在恰好 $j-1$ 个环已经形成之后的操作）。在第 $j$ 步，自由端数为 $2(100 - (j-1)) = 2n$ 个，其中 $n = 100 - j + 1$ 是尚未形成环的鞋带数。选择第一个自由端有 $2n$ 种方式，选择第二个有 $2n-1$ 种方式。形成闭环 = 选到了同一根鞋带的两个自由端：共有 $n$ 根散鞋带，每根贡献 $\binom{2}{2} = 1$ 种方式 = 实际上就是选同一个鞋带的两端。

$$P(\text{第 } j \text{ 步形成闭环}) = \frac{n}{2n(2n-1)/2} = \frac{1}{2n-1}$$

因此期望环数 = $\sum_{n=1}^{100} \frac{1}{2n-1} = 1 + \frac{1}{3} + \frac{1}{5} + \cdots + \frac{1}{199}$。

不用计算具体数值（约等于 3.28）。关键在于：**这个问题用指示变量法几乎"免费"解决——如果不用线性期望，你需要分析所有可能的鞋带连接图结构，那将极其复杂。**

**训练点：** 指示变量法用于"过程的每一步是否创造了一个环"。线性期望让你免于处理图结构的复杂性。

**题 2（SP4-1.4 | 反直觉的期望）**

是否存在离散随机变量 $X$ 和 $Y$，使得 $E(X) > 100E(Y)$，但 $Y > X$ 的概率至少为 0.99？如果存在，给出一个例子。

**▼ 答案与解析**

存在。构造：

- $X$：以概率 $1/100$ 取值为 $10^6$，以概率 $99/100$ 取值为 $0$
- $Y = 1$（常数）

则：
- $E(X) = 10^6 \times 0.01 + 0 \times 0.99 = 10000$
- $E(Y) = 1$
- $E(X) = 10000 \times E(Y)$，远超 100 倍

而 $P(Y > X) = P(X = 0) = 0.99$。

**洞察：** 这个例子和 St. Petersburg 悖论（第三章题 7）有相同的思想内核——**期望可以被小概率的极端值主导，从而与任何一次观测的"典型结果"相去万里。** 在 99% 的情况下，$Y$（值 1）大于 $X$（值 0）。但剩下那 1% 的巨值（$10^6$）把期望拉到了一个荒谬的高度。

这告诉我们：期望作为"中心趋势的度量"是有局限的——当分布高度偏斜或有重尾时，期望和"典型值"之间的鸿沟可以任意大。

**训练点：** 期望描述的是"长期平均"，不是"一次观测中最可能出现的结果"。不要混淆期望和众数。

**题 3（SP4-1.5 | 尾和公式——CDF 与期望的桥梁）**

设 $X$ 是取值为非负整数的随机变量，CDF 为 $F(x) = P(X \leq x)$。证明：

$$E(X) = \sum_{n=0}^{\infty} (1 - F(n)) = \sum_{n=0}^{\infty} P(X > n)$$

**▼ 答案与解析**

注意到 $1 - F(n) = P(X > n) = \sum_{k=n+1}^{\infty} P(X = k)$。所以：

$$\sum_{n=0}^{\infty} P(X > n) = \sum_{n=0}^{\infty} \sum_{k=n+1}^{\infty} P(X = k)$$

交换求和次序：对固定的 $k$，$P(X = k)$ 出现在所有满足 $n < k$（即 $n = 0, 1, \ldots, k-1$）的项中——恰好 $k$ 次。所以：

$$\sum_{n=0}^{\infty} P(X > n) = \sum_{k=1}^{\infty} k \cdot P(X = k) = E(X)$$

**直觉：** $E(X) = \sum P(X > n)$ 提供了计算期望的另一种方式——不是按"每个可能取值 × 其概率"来加，而是按"$X$ 大于 $n$ 的概率"在 $n = 0, 1, 2, \ldots$ 上累积。对于"等待时间"类问题，$P(X > n)$ 通常比 $P(X = n)$ 更容易计算，这个公式可以大大简化期望的计算。

**训练点：** 尾和公式是 Geometric 期望计算的另一种路径——$P(X > n) = (1-p)^{n+1}$（注意 Blitzstein 的 Geometric 定义从 0 开始），$\sum_{n=0}^{\infty} (1-p)^{n+1} = \frac{1-p}{p}$，一步到位。

**题 4（SP4-1.6 | 求职者问题——期望无穷大）**

求职者 $C_1, C_2, \ldots$ 依次面试。设 $X$ 为第一个比 $C_1$ 优秀的求职者的序号（如果 $C_1$ 恰好是最优秀的，则 $X = \infty$）。假设求职者的能力排名是完全随机的（所有排列等可能）。求 $E(X)$。

**▼ 答案与解析**

首先计算 $P(X > n)$：$X > n$ 意味着在前 $n$ 个求职者中，$C_1$ 是最优秀的（否则在 $n$ 之前就已经遇到了比 $C_1$ 更优秀的人）。在所有排列中，前 $n$ 人中 $C_1$ 排名第一的概率为 $1/n$（对称性——$C_1$ 在前 $n$ 人的任意排列中，排第一的概率是 $1/n$）。

所以对 $n \geq 1$：$P(X > n) = 1/n$。对 $n = 0$：$P(X > 0) = P(X \geq 1) = 1$（第一个求职者总是 $C_1$ 自己，不会在它之前遇到更好的）。

用尾和公式（题 3）：

$$E(X) = \sum_{n=0}^{\infty} P(X > n) = 1 + \sum_{n=1}^{\infty} \frac{1}{n} = \infty$$

**期望是无穷大！** 调和级数 $\sum 1/n$ 发散。

**洞察：** 这个结果与 St. Petersburg 悖论异曲同工——虽然最终一定会遇到比 $C_1$ 更优秀的人（只要候选人无穷多），但你**等待的期望时间是无穷大**。在实际中候选人数是有限的（设为 $N$），则 $E(X) = 1 + \sum_{n=1}^{N-1} 1/n \approx 1 + \ln(N-1) + \gamma$（$\gamma \approx 0.577$ 是 Euler-Mascheroni 常数），增长非常缓慢。

**训练点：** 尾和公式 + 对称性 = 绕开 PMF 直接求期望。调和级数的发散性在概率论中反复出现（等待时间、记录值等问题）。

### 第二组：协方差与相关系数

---

**题 5（协方差为零但不独立——经典构造）**

设 $X \sim \text{Uniform}\{-1, 0, 1\}$（取 $-1, 0, 1$ 各以概率 $1/3$）。令 $Y = X^2$。

(a) 证明 $\text{Cov}(X, Y) = 0$。
(b) 证明 $X$ 和 $Y$ 不独立。
(c) 请解释：既然 $Y$ 完全由 $X$ 决定，为什么协方差还会是 0？

**▼ 答案与解析**

(a) $E(X) = \frac{1}{3}(-1 + 0 + 1) = 0$。$E(XY) = E(X \cdot X^2) = E(X^3)$。由于 $X \in \{-1, 0, 1\}$，$X^3 = X$，所以 $E(X^3) = E(X) = 0$。

$$\text{Cov}(X, Y) = E(XY) - E(X)E(Y) = 0 - 0 \cdot E(Y) = 0$$

(b) $P(X = 0, Y = 0) = P(X = 0) = 1/3$。而 $P(X = 0)P(Y = 0) = \frac{1}{3} \times \frac{1}{3} = \frac{1}{9}$。两者不相等，所以不独立。

(c) 协方差衡量的是**线性**关系。$Y = X^2$ 是完美的平方关系（对称的 U 形）。$X$ 的分布关于 0 对称，所以 $X$ 的正偏差和负偏差对 $E[(X-E(X))(Y-E(Y))]$ 的贡献恰好互相抵消——这不是因为没有关系，而是因为关系的形状（对称抛物线）在协方差这个线性度量下"看不见"。

**训练点：** $\text{Cov}(X, Y) = 0$（不相关）不等于独立。相关系数/协方差只能检测线性依赖模式。判断独立性需要检查 $P(X=x, Y=y) = P(X=x)P(Y=y)$ 对所有 $x, y$ 成立。

---

**题 6（相关系数的性质——边界条件的证明）**

证明：$-1 \leq \text{Corr}(X, Y) \leq 1$。

提示：考虑 $\text{Var}\left(\frac{X}{\sigma_X} \pm \frac{Y}{\sigma_Y}\right)$，其中 $\sigma_X = \text{SD}(X)$，$\sigma_Y = \text{SD}(Y)$。

**▼ 答案与解析**

方差必须非负。展开：

$$
\begin{aligned}
\text{Var}\left(\frac{X}{\sigma_X} + \frac{Y}{\sigma_Y}\right) &= \frac{\text{Var}(X)}{\sigma_X^2} + \frac{\text{Var}(Y)}{\sigma_Y^2} + 2\frac{\text{Cov}(X, Y)}{\sigma_X \sigma_Y} \\
&= 1 + 1 + 2\text{Corr}(X, Y) \\
&= 2 + 2\text{Corr}(X, Y) \geq 0
\end{aligned}
$$

所以 $\text{Corr}(X, Y) \geq -1$。

同理，$\text{Var}\left(\frac{X}{\sigma_X} - \frac{Y}{\sigma_Y}\right) = 2 - 2\text{Corr}(X, Y) \geq 0$，所以 $\text{Corr}(X, Y) \leq 1$。

**更重要的是等号条件：** $\text{Corr}(X, Y) = 1$ 当且仅当 $\text{Var}\left(\frac{X}{\sigma_X} - \frac{Y}{\sigma_Y}\right) = 0$，即 $\frac{X}{\sigma_X} - \frac{Y}{\sigma_Y}$ 几乎必然为常数——这意味着 $Y = aX + b$ 且 $a > 0$。同样，$\text{Corr} = -1$ 对应 $Y = aX + b$ 且 $a < 0$。

**训练点：** 这个证明展示了方差的非负性是很多概率不等式的来源（本质上是 Cauchy-Schwarz 不等式在概率空间的实例）。标准化的技巧（除以标准差得到无量纲量）也是一个重要的方法论。

### 第三组：条件期望与 Adam's Law

---

**题 7（Adam's Law——赌徒破产的期望时间）**

一个赌徒进行公平赌博（$p = 0.5$），每次赢或输 1 元。他从 $k$ 元开始（$0 < k < N$），一旦达到 $N$ 或 0 就停止。设 $D_k$ 为从 $k$ 元开始直到停止的期望步数。用 Adam's Law 推导 $D_k$ 满足的递推关系，并验证 $D_k = k(N-k)$。

**▼ 答案与解析**

条件于第一步（赢或输）：

$$D_k = 1 + \frac{1}{2} D_{k+1} + \frac{1}{2} D_{k-1}$$

其中 "1" 表示第一步本身，$\frac{1}{2}D_{k+1}$ 是第一步赢后从 $k+1$ 走到终点的期望步数。边界条件：$D_0 = D_N = 0$。

整理为差分方程：$D_{k+1} - 2D_k + D_{k-1} = -2$。

验证 $D_k = k(N-k)$ 满足该方程（代入展开，交叉项均互相抵消，只留下 $-2$）。

$$D_k = k(N-k)$$

**洞察：** 当 $k = N/2$（正中间）时，$D_k = N^2/4$。如果目标是 $N = 100$（从 50 元开始），期望需要 2500 步。公平赌博的期望时间是二次增长的——这在直觉上容易被严重低估。

**训练点：** Adam's Law 在序贯过程中的经典用法——条件于"第一步"，把动态过程拆解为"这步的成本 + 从新状态出发的期望"。这个"一步条件化"的技巧在马尔可夫链、随机游走等问题中反复使用。

**题 8（Adam's Law + Poisson 稀疏性质——随机个数和）**

设 $N \sim \text{Pois}(\lambda)$，且 $X_1, X_2, \ldots \overset{i.i.d.}{\sim} \text{Bern}(p)$，与 $N$ 独立。定义 $S = \sum_{j=1}^N X_j$（随机个 Bernoulli 的和）。

(a) 用 Adam's Law 求 $E(S)$。
(b) $S$ 服从什么分布？（提示：考虑给定 $N$ 后的条件分布。）

**▼ 答案与解析**

(a) 条件于 $N$：

$$E(S) = \sum_{n=0}^\infty E(S \mid N = n) \cdot P(N = n)$$

当 $N = n$ 时，$S \mid N=n \sim \text{Bin}(n, p)$，所以 $E(S \mid N = n) = np$。

$$E(S) = \sum_{n=0}^\infty np \cdot \frac{e^{-\lambda}\lambda^n}{n!} = p \cdot E(N) = p\lambda$$

(b) $S \sim \text{Pois}(\lambda p)$。这被称为 **Poisson 稀疏 (thinning)** 性质。

证明思路：$S$ 只能取非负整数，且：
$$P(S = k) = \sum_{n=k}^\infty P(S = k \mid N = n) P(N = n) = \sum_{n=k}^\infty \binom{n}{k} p^k (1-p)^{n-k} \cdot \frac{e^{-\lambda}\lambda^n}{n!}$$

通过代数化简（利用 $\lambda^n = \lambda^k \cdot \lambda^{n-k}$ 和 $\frac{1}{n!} = \frac{1}{k!(n-k)!} \cdot \frac{k!}{n!}$ 再整理），得到 Poisson($\lambda p$) 的 PMF。

**直觉：** 如果事件的发生按照 $\text{Pois}(\lambda)$ 到达，而每个事件以独立概率 $p$ 被"标记"为"成功"，那么成功事件的发生仍然服从 Poisson，速率被稀释为 $\lambda p$。这个性质在排队论、保险精算、网络流量分析中有广泛应用。

**训练点：** Adam's Law 结合条件分布 (conditional distribution) 破解"随机个数和"问题。Poisson 的稀疏性质是它区别于其他分布的一个独特且优美的特征。

## 七、本章的核心脉络（回顾）

第四章可以在三个层次上理解：

**第一层——期望与方差的深化。** 从第三章的"定义和基本计算"升级到对线性性质、方差结构的系统理解。你不再只看一个随机变量，而是开始考虑多个变量之间的代数和关系。

**第二层——协方差与相关系数。** 引入衡量变量间关系的定量工具。协方差告诉你"它们是否同步波动"，相关系数将这种关系的强度标准化到 $[-1, 1]$。但你必须警惕：零相关不等于独立——它只意味着没有线性关系。

**第三层——条件期望与 Adam's Law。** 这是第二章的"条件概率思维"在期望领域的延伸。条件期望 $E(X \mid A)$ 告诉你"假设我们知道 $A$ 发生了，$X$ 的均值是多少"。Adam's Law 告诉你如何将条件期望按概率加权求和来得到无条件期望。这是处理分阶段问题、序贯问题、分层抽样问题的核心工具。

如果你只从这一章带走三样东西：
- **期望的线性性质不需要独立性**——这也许是概率论中最实用的定理，它让你用指示变量法解决几乎所有计数类期望问题
- **$\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X,Y)$**——方差是否可加取决于协方差，而协方差衡量的是线性同步程度
- **$E(X) = \sum E(X \mid B_j) P(B_j)$**（Adam's Law）——条件于合适的划分，把复杂期望分解为简单期望的加权平均
