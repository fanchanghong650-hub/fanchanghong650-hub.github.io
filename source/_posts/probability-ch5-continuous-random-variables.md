## 为什么第五章是全书最重要的"范式转换"

前三章你学会了用概率质量函数（PMF）描述离散随机变量——骰子的点数、硬币的朝向、一年中发生的事故次数。这些都是**可数的**结果。

但现实世界充满了不可数的量：等待时间、测量误差、身高体重、降水量……它们不能用一个列表穷举。第五章教你用**概率密度函数（PDF）**来描述这些连续随机变量。

从离散到连续的过渡，是概率论中最重要的一次思维升级：

> PMF 给出的是**概率值**（每个点的质量）。PDF 给出的是**概率密度**（单位长度上的"浓度"）——单个点上的概率永远是零，概率只存在于**区间**中。

理解这一点是掌握整个后半本书的前提。如果你不理解 PDF 中的 $f(x)$ 为什么可以大于 1，你会在联合分布（第七章）和变换（第八章）中反复碰壁。

## 前置知识

- 第四章期望的基本概念（$E(X)$ 的定义、LOTUS 的离散版本）
- 基本的微积分知识：定积分、求导、指数函数与对数函数
- 第三章常见离散分布（二项、几何、泊松）的 PMF——很多连续分布是它们的自然极限
- 极限的基本直觉（如 $\lim_{n\to\infty} (1 + x/n)^n = e^x$）

## 一、PDF 与 CDF：连续世界的两把钥匙

### 1.1 PDF 不是概率

**定义**：一个随机变量 $X$ 有概率密度函数（PDF）$f_X(x)$，如果对 $\mathbb{R}$ 的任意（Borel）子集 $B$：

$$P(X \in B) = \int_B f_X(x) \, dx$$

PDF 必须满足：
- **非负性**：$f_X(x) \geq 0$ 对**所有** $x$
- **归一化**：$\int_{-\infty}^{\infty} f_X(x) \, dx = 1$

**最重要的直觉**：$f_X(3) = 2$ 是合法的——只要它在 $[3, 3.01]$ 上的积分约等于 $2 \times 0.01 = 0.02$。PDF 本身不是概率，它是**概率密度**。把一根有质量的细棒想像成一维物体——棒上每一点的质量微元是密度乘以 $dx$。密度可以大于 1，只要总质量 = 1。

### 1.2 CDF：连通离散与连续的桥梁

无论 $X$ 是离散还是连续，CDF（累积分布函数）都有相同的定义：

$$F_X(x) = P(X \leq x)$$

在连续情况下：

$$F_X(x) = \int_{-\infty}^x f_X(t) \, dt$$

且微积分基本定理告诉我们：

$$f_X(x) = F_X'(x)$$

（在 $f_X$ 连续的点上成立。）

**CDF 的三个关键性质**（对连续分布）：
1. $F_X$ 是**连续**的（离散情况下是阶梯函数）
2. $F_X$ 是**单调不减**的
3. $\lim_{x\to -\infty} F_X(x) = 0$，$\lim_{x\to \infty} F_X(x) = 1$

> **离散 vs 连续的实质区别**：
> - 离散：$F_X$ 有跳跃，跳跃的大小 = 该点的概率质量
> - 连续：$F_X$ 光滑，任何单点的概率 = 0

### 1.3 连续情况下的期望与 LOTUS

连续随机变量的期望定义：

$$E(X) = \int_{-\infty}^{\infty} x \cdot f_X(x) \, dx$$

**LOTUS（连续版）**——这可能是第五章最实用的定理：

$$E(g(X)) = \int_{-\infty}^{\infty} g(x) \cdot f_X(x) \, dx$$

你**不需要**先求 $g(X)$ 的分布。直接把 $g(x)$ 乘上 $f_X(x)$ 积分即可。

这是计算 $E(X^2)$、$E(e^X)$、$E(|X|)$ 等一切函数期望的统一方法。

## 二、均匀分布：最简单的连续分布

### 2.1 定义与直觉

$X \sim \text{Unif}(a, b)$ 表示 $X$ 在区间 $[a, b]$ 上均匀分布：

$$f_X(x) = \frac{1}{b-a}, \quad a < x < b$$

$$F_X(x) = \frac{x-a}{b-a}, \quad a \leq x \leq b$$

$$E(X) = \frac{a+b}{2}, \quad \text{Var}(X) = \frac{(b-a)^2}{12}$$

$\text{Unif}(0,1)$ 被称为**标准均匀分布**，是整个连续分布体系的"原子单位"。

### 2.2 均匀性的普适原理（Universality of the Uniform）

这是第五章最深刻的洞察之一，值得用一整节来讲解。

**定理**：设 $X$ 是一个连续随机变量，其 CDF $F$ 是严格递增的。则：

$$U = F(X) \sim \text{Unif}(0,1)$$

也就是说——**任何连续随机变量，经过自己的 CDF 变换后，都变成标准均匀分布**。

**直觉**：CDF 告诉你"$X$ 在哪个百分位上"。$F(X)$ 就是"$X$ 在所有可能取值中排在什么位置"——这个位置自然均匀分布在 $[0,1]$ 上。

### 2.3 逆变换采样：从均匀到任意分布

把上述定理反过来读：

$$X = F^{-1}(U)$$

其中 $U \sim \text{Unif}(0,1)$，$F^{-1}$ 是目标分布的 CDF 反函数。

这告诉你**如何从均匀随机数生成任意连续分布**：
1. 生成 $U \sim \text{Unif}(0,1)$
2. 计算 $X = F^{-1}(U)$
3. $X$ 服从目标分布

这是所有统计软件抽样算法的核心思想。

**例子**：从 $U \sim \text{Unif}(0,1)$ 生成指数分布 $X \sim \text{Expo}(\lambda)$。

指数分布的 CDF 为 $F(x) = 1 - e^{-\lambda x}$，其反函数为 $F^{-1}(u) = -\frac{1}{\lambda}\ln(1-u)$。

由于 $1-U$ 也是 $\text{Unif}(0,1)$，可以简化为 $X = -\frac{1}{\lambda}\ln U$。

这就是为什么统计软件能用一个 `random()` 函数生成任何分布——背后是均匀性的普适原理。

## 三、正态分布：概率论的中心角色

### 3.1 标准正态分布

标准正态分布 $Z \sim \mathcal{N}(0,1)$ 的 PDF 是：

$$\varphi(z) = \frac{1}{\sqrt{2\pi}} e^{-z^2/2}, \quad z \in \mathbb{R}$$

这个钟形曲线是概率论中最著名的函数。它的积分没有闭式——这解释了为什么我们需要标准正态表（或软件）。

CDF 记为 $\Phi(z) = P(Z \leq z)$。几个关键值必须记住：
- $\Phi(-1) \approx 0.159$，$\Phi(0) = 0.5$，$\Phi(1) \approx 0.841$
- $P(|Z| \leq 1) \approx 0.683$，$P(|Z| \leq 2) \approx 0.954$，$P(|Z| \leq 3) \approx 0.997$

最后一个就是著名的**68-95-99.7 规则**。

### 3.2 一般正态分布

通过位置-尺度变换从标准正态得到：

$$X = \mu + \sigma Z \sim \mathcal{N}(\mu, \sigma^2)$$

其 PDF 为：

$$f_X(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-(x-\mu)^2/(2\sigma^2)}$$

**关键技巧——标准化**：将 $X \sim \mathcal{N}(\mu, \sigma^2)$ 转化为 $Z = \frac{X-\mu}{\sigma} \sim \mathcal{N}(0,1)$：

$$P(X \leq x) = P\left(Z \leq \frac{x-\mu}{\sigma}\right) = \Phi\left(\frac{x-\mu}{\sigma}\right)$$

### 3.3 正态分布的对称性

标准正态 PDF 满足 $\varphi(-z) = \varphi(z)$。由此可得：

$$\Phi(-z) = 1 - \Phi(z)$$

这一对称性可以帮你只用一半的标准正态表。（实际上 $P(|Z| > z) = 2\Phi(-z) = 2(1 - \Phi(z))$。）

### 3.4 正态近似：CLT 的预告

虽然中心极限定理在第十章才正式登场，但第五章已经暗示了它的存在：许多独立随机变量的和在适当标准化后趋近正态分布。这就是为什么正态分布出现在测量误差、生物特征、金融收益率等几乎所有连续数据中。

## 四、指数分布：无记忆的时间

### 4.1 定义与基本性质

$X \sim \text{Expo}(\lambda)$ 的 PDF 为：

$$f_X(x) = \lambda e^{-\lambda x}, \quad x > 0$$

$$F_X(x) = 1 - e^{-\lambda x}, \quad x > 0$$

$$E(X) = \frac{1}{\lambda}, \quad \text{Var}(X) = \frac{1}{\lambda^2}$$

参数 $\lambda$ 是**速率**（rate）——$\lambda$ 越大，等待时间越短。

### 4.2 无记忆性：指数分布的定义性特征

指数分布是**唯一**具有无记忆性的连续分布：

$$P(X > s + t \mid X > s) = P(X > t)$$

**直觉**：如果你已经等了一小时公交车还没来，那么"还要再等多久"的分布和"从零开始等"的分布完全一样。公交车不记得你已经等了一小时——它没有变老。

数学验证很简单：

$$P(X > s+t \mid X > s) = \frac{P(X > s+t)}{P(X > s)} = \frac{e^{-\lambda(s+t)}}{e^{-\lambda s}} = e^{-\lambda t} = P(X > t)$$

> **指数分布的无记忆性是离散几何分布无记忆性的连续类比。** $n$ 次失败后下一次试验成功的概率还是一样——骰子不记得之前的失败。

### 4.3 泊松过程：离散与连续的桥梁

指数分布和泊松分布描述的是**同一过程**的两个侧面：

- **泊松分布**：固定时间窗口内，事件发生的**次数**
- **指数分布**：连续两次事件之间的**等待时间**

**泊松过程的正式定义**：事件以速率 $\lambda$ 发生，满足：
1. 在长度为 $t$ 的区间内，事件发生次数 $\sim \text{Pois}(\lambda t)$
2. 不相交区间内的事件次数独立
3. 两次事件之间的等待时间 $\stackrel{iid}{\sim} \text{Expo}(\lambda)$

**计数-时间对偶性（Count-Time Duality）**——这是使用泊松过程解题的关键：

$$P(T_n > t) = P(N_t < n)$$

即"第 $n$ 次到达的时刻超过 $t$"等价于"在 $t$ 时刻之前到达次数少于 $n$"。这一转换把指数分布（等待时间）的问题变成了泊松分布（计数）的问题，或反之。

## 精选习题

### 第一组：PDF/CDF 基本功 + 均匀分布

**题 1（SP5 | 均匀到指数 —— 逆变换采样）**

设 $U \sim \text{Unif}(0,1)$，$\lambda > 0$。

(a) 证明 $X = -\frac{1}{\lambda}\ln(1-U) \sim \text{Expo}(\lambda)$。
(b) 由此推导指数分布的期望和方差。

**▼ 答案与解析**

(a) 使用 CDF 方法。对任意 $x > 0$：

$$P(X \leq x) = P\left(-\frac{1}{\lambda}\ln(1-U) \leq x\right) = P(\ln(1-U) \geq -\lambda x) = P(1-U \geq e^{-\lambda x}) = P(U \leq 1 - e^{-\lambda x})$$

由于 $U \sim \text{Unif}(0,1)$，$P(U \leq t) = t$（$0 \leq t \leq 1$），所以：

$$P(X \leq x) = 1 - e^{-\lambda x}$$

这正是 $\text{Expo}(\lambda)$ 的 CDF。证毕。

(b) 由 $X = -\frac{1}{\lambda}\ln(1-U)$，且 $\ln(1-U)$ 的分布与 $\ln U$ 相同（对称性）：

$$E(X) = E\left(-\frac{1}{\lambda}\ln U\right) = -\frac{1}{\lambda} \int_0^1 \ln u \, du = \frac{1}{\lambda}$$

$$E(X^2) = \frac{1}{\lambda^2} \int_0^1 (\ln u)^2 \, du = \frac{2}{\lambda^2}$$

$$\text{Var}(X) = E(X^2) - [E(X)]^2 = \frac{2}{\lambda^2} - \frac{1}{\lambda^2} = \frac{1}{\lambda^2}$$

**洞察**：这道题展示了"均匀性的普适原理"最经典的应用——逆变换采样。背后的逻辑是：均匀随机数 + 合适的逆 CDF 变换 = 任意分布。现代统计软件中用 `random()` 生成一切分布，用的就是这个原理。

**训练点**：CDF 方法推导变换后的分布 | 逆变换采样的理解 | 积分计算期望

**题 2（教材 5.5 | 随机圆的面积）**

圆的半径 $R \sim \text{Unif}(0,1)$。令 $A = \pi R^2$ 为圆的面积。

(a) 求 $A$ 的 CDF 和 PDF。
(b) 求 $E(A)$ 和 $\text{Var}(A)$。$A$ 的分布是否也是均匀的？

**▼ 答案与解析**

(a) $A$ 的取值范围是 $[0, \pi]$。对 $0 \leq a \leq \pi$：

$$F_A(a) = P(A \leq a) = P(\pi R^2 \leq a) = P\left(R \leq \sqrt{\frac{a}{\pi}}\right) = \sqrt{\frac{a}{\pi}}$$

（因为 $R \sim \text{Unif}(0,1)$，$P(R \leq r) = r$。）

求导得 PDF：$f_A(a) = \frac{1}{2\sqrt{\pi a}}$，$0 < a < \pi$。

(b) 方法 1（用 $R$ 的分布 + LOTUS）：

$$E(A) = E(\pi R^2) = \pi \int_0^1 r^2 \cdot 1 \, dr = \pi \cdot \frac{1}{3} = \frac{\pi}{3}$$

$$E(A^2) = \pi^2 E(R^4) = \pi^2 \int_0^1 r^4 \, dr = \frac{\pi^2}{5}$$

$$\text{Var}(A) = \frac{\pi^2}{5} - \frac{\pi^2}{9} = \frac{4\pi^2}{45}$$

方法 2（直接用 $A$ 的 PDF）：$E(A) = \int_0^\pi a \cdot \frac{1}{2\sqrt{\pi a}} \, da = \frac{\pi}{3}$。结果一致。

**$A$ 不是均匀分布**——它的 PDF $f_A(a) = \frac{1}{2\sqrt{\pi a}}$ 在 $a=0$ 附近很大，在 $a=\pi$ 附近很小。这说明一个关键点：**均匀分布的（非线性）变换通常不产生均匀分布**。

**洞察**：这题最关键的教训是用 LOTUS 绕开变量替换——直接用 $A$ 的表达式在 $R$ 的分布上积分，比先求 $A$ 的 PDF 再积分要简单得多。LOTUS 的存在意义就在此。

**训练点**：CDF 方法求变换后分布 | LOTUS 优选路径的判断 | 函数变换一般不保均匀性

**题 3（教材 5.14 | $n$ 个均匀分布的最大值）**

设 $U_1, U_2, \ldots, U_n \stackrel{iid}{\sim} \text{Unif}(0,1)$，$X = \max(U_1, \ldots, U_n)$。

(a) 求 $X$ 的 CDF 和 PDF。
(b) 求 $E(X)$ 并解释 $n \to \infty$ 时 $X$ 的行为。

**▼ 答案与解析**

(a) $X \leq x$ 当且仅当**所有** $U_i \leq x$。由独立性：

$$F_X(x) = P(X \leq x) = P(U_1 \leq x, \ldots, U_n \leq x) = \prod_{i=1}^n P(U_i \leq x) = x^n, \quad 0 \leq x \leq 1$$

求导得 $f_X(x) = n x^{n-1}$，$0 < x < 1$。

(b) $E(X) = \int_0^1 x \cdot n x^{n-1} \, dx = n \int_0^1 x^n \, dx = \frac{n}{n+1}$。

当 $n \to \infty$ 时，$E(X) \to 1$。且 $\text{Var}(X) = \frac{n}{(n+1)^2(n+2)} \to 0$。

**直觉**：样本量 $n$ 越大，最大值越确定地靠近 1。取 $n=100$，$E(X) \approx 0.99$——100 个均匀随机数中最大的那个几乎必然非常接近上界 1。这是"极值理论"最简单的例子。

**洞察**：最大值（和最小值）统计量的分布公式是解题的常用工具。核心技巧是：$P(\max \leq x) = P(\text{所有} \leq x)$ 和 $P(\min > x) = P(\text{所有} > x)$——利用独立性把联合事件分解为乘积。

**训练点**：最大值统计量的 CDF 公式 | 独立性条件下联合概率的分解 | $n \to \infty$ 极限行为

### 第二组：正态分布与对称性

**题 4（SP5 | 对数正态分布）**

设 $X \sim \mathcal{N}(\mu, \sigma^2)$，定义 $Y = e^X$（$Y$ 称为对数正态分布）。

(a) 求 $Y$ 的 CDF。
(b) 求 $Y$ 的 PDF。

**▼ 答案与解析**

(a) 对 $y > 0$：

$$F_Y(y) = P(Y \leq y) = P(e^X \leq y) = P(X \leq \ln y) = \Phi\left(\frac{\ln y - \mu}{\sigma}\right)$$

（如果 $y \leq 0$，$F_Y(y) = 0$，因为指数函数恒正。）

(b) 对 CDF 求导得 PDF。使用链式法则：

$$f_Y(y) = \frac{d}{dy} \Phi\left(\frac{\ln y - \mu}{\sigma}\right) = \frac{1}{y\sigma} \cdot \varphi\left(\frac{\ln y - \mu}{\sigma}\right)$$

$$= \frac{1}{y\sigma\sqrt{2\pi}} \exp\left(-\frac{(\ln y - \mu)^2}{2\sigma^2}\right), \quad y > 0$$

**洞察**：对数正态分布广泛用于金融建模——股票价格、资产收益率、保险索赔额。原因是：收益率（百分比变化）的乘积取对数后变成和，而和由 CLT 趋近正态——所以原始价格趋近对数正态。

**训练点**：CDF 方法 | 正态标准化 | 链式法则求导 | 对数正态的应用背景

**题 5（SP5 | $\Phi(Z)$ 的期望——均匀性普适原理的巧用）**

设 $Z \sim \mathcal{N}(0,1)$。求 $E(\Phi(Z))$，其中 $\Phi$ 是标准正态的 CDF。

**▼ 答案与解析**

方法 1（直接 LOTUS）——可行但需要计算一个并不显然的积分：

$$E(\Phi(Z)) = \int_{-\infty}^{\infty} \Phi(z) \cdot \frac{1}{\sqrt{2\pi}} e^{-z^2/2} \, dz$$

方法 2（均匀性普适原理）——绕过积分，一秒钟搞定：

根据**均匀性的普适原理**：若 $F$ 是 $Z$ 的 CDF 且严格递增，则 $F(Z) \sim \text{Unif}(0,1)$。

这里 $F = \Phi$，$Z \sim \mathcal{N}(0,1)$，所以 $\Phi(Z) \sim \text{Unif}(0,1)$。

于是 $E(\Phi(Z))$ 就是一个标准均匀分布的期望 = $\frac{1}{2}$。

**完全不需要积分。**

**洞察**：这题展示了均匀性普适原理的威力。当你看到形如 $F_X(X)$ 的表达式时（$F_X$ 是 $X$ 的 CDF），立刻反应：它服从 $\text{Unif}(0,1)$。这个技巧在统计推断中极其常用——p 值在零假设下服从均匀分布，就是同一个原理。

**训练点**：均匀性普适原理的识别与应用 | 绕开 LOTUS 的巧算技巧

**题 6（教材 5.25 | 两个独立正态的大小比较）**

设 $X \sim \mathcal{N}(\mu_X, \sigma_X^2)$ 与 $Y \sim \mathcal{N}(\mu_Y, \sigma_Y^2)$ 独立。求 $P(X < Y)$。

**▼ 答案与解析**

关键观察：$X - Y$ 也是正态的——正态分布在独立线性组合下保持正态：

$$X - Y \sim \mathcal{N}(\mu_X - \mu_Y, \sigma_X^2 + \sigma_Y^2)$$

（方差相加是因为 $\text{Var}(X - Y) = \text{Var}(X) + \text{Var}(-Y) = \sigma_X^2 + \sigma_Y^2$。）

于是：

$$P(X < Y) = P(X - Y < 0) = \Phi\left(\frac{0 - (\mu_X - \mu_Y)}{\sqrt{\sigma_X^2 + \sigma_Y^2}}\right) = \Phi\left(\frac{\mu_Y - \mu_X}{\sqrt{\sigma_X^2 + \sigma_Y^2}}\right)$$

**特殊情形**：如果 $\mu_X = \mu_Y$ 且 $\sigma_X = \sigma_Y$，则 $P(X < Y) = \Phi(0) = \frac{1}{2}$——由对称性几乎显而易见。

**洞察**：这是 $P(X < Y)$ 这类问题的标准解法——不要比较 $X$ 和 $Y$，而是考察它们的差 $X-Y$，将其转化为单一正态变量的概率计算。正态分布在加法下封闭——这是它最重要的代数性质之一。

**训练点**：独立正态差仍为正态 | 标准化技巧 | $P(X<Y)$ 的标准处理范式

### 第三组：指数分布、无记忆性与泊松过程

**题 7（教材 5.37 | 指数分布的无记忆性）**

放射性粒子的寿命 $T \sim \text{Expo}(\lambda)$（$\lambda$ 为衰变速率）。

(a) 证明：$P(T > s + t \mid T > s) = P(T > t)$（无记忆性）。
(b) 半衰期 $t_{1/2}$ 满足 $P(T > t_{1/2}) = 1/2$。用 $\lambda$ 表达 $t_{1/2}$。
(c) 设 $n$ 个粒子的寿命 $T_1, \ldots, T_n \stackrel{iid}{\sim} \text{Expo}(\lambda)$。第一个衰变的粒子的等待时间是什么分布？

**▼ 答案与解析**

(a) $P(T > x) = e^{-\lambda x}$（指数分布的生存函数）。因此：

$$P(T > s+t \mid T > s) = \frac{P(T > s+t)}{P(T > s)} = \frac{e^{-\lambda(s+t)}}{e^{-\lambda s}} = e^{-\lambda t} = P(T > t) \quad \square$$

(b) $P(T > t_{1/2}) = e^{-\lambda t_{1/2}} = 1/2$，所以 $t_{1/2} = \frac{\ln 2}{\lambda}$。

(c) 设 $M = \min(T_1, \ldots, T_n)$ 为第一个粒子衰变的时间。

$$P(M > t) = P(\text{所有 } T_i > t) = \prod_{i=1}^n P(T_i > t) = (e^{-\lambda t})^n = e^{-n\lambda t}$$

所以 $M \sim \text{Expo}(n\lambda)$——**$n$ 个独立的同速率指数分布的最小值服从速率加倍的指数分布**。

**直觉**：$n$ 个独立衰变的粒子，第一个衰变的等待速率是单个粒子的 $n$ 倍——因为"竞争"使衰变更快发生。

**洞察**：(c) 的结果可以推广：如果 $T_i \stackrel{iid}{\sim} \text{Expo}(\lambda_i)$ 独立但**不同速率**，则 $M \sim \text{Expo}(\lambda_1 + \cdots + \lambda_n)$。这是泊松过程"叠加性质"在指数分布侧的体现。

**训练点**：无记忆性的严格证明 | 指数分布生存函数的使用 | 独立指数的最小值

**题 8（教材 5.36 | 邮局的两个窗口）**

邮局有两个职员，三个顾客 $A, B, C$ 同时到达。$A$ 和 $B$ 直接走向两个窗口（每个窗口服务一个顾客），$C$ 等待第一个完成的顾客。服务时间 $\stackrel{iid}{\sim} \text{Expo}(\lambda)$，顾客之间独立。

(a) 求 $C$ **不是**最后一个离开的概率。
(b) 求 $C$ 在邮局的总等待时间（服务开始直到离开）的期望。

**▼ 答案与解析**

(a) 由指数分布的无记忆性，当第一个窗口完成服务时，第二个窗口的剩余服务时间仍服从 $\text{Expo}(\lambda)$——和刚开始时一样。

此时，第一个完成服务的顾客离开，$C$ 接管那个窗口。局面变成：$C$ 和一个仍在被服务的顾客**同时开始**被服务，两个服务时间 $\stackrel{iid}{\sim} \text{Expo}(\lambda)$。

由对称性，$C$ 先于另一个人完成服务的概率为 $P(C \text{ not last}) = \frac{1}{2}$。

（更仔细的分析：设第一个完成的是 $A$，耗时 $T_A$。$C$ 接管 $A$ 的窗口，服务时间 $T_C \sim \text{Expo}(\lambda)$。$B$ 的剩余服务时间 $T_B' \sim \text{Expo}(\lambda)$ 由无记忆性保证。$P(C < B) = P(T_C < T_B') = \frac{\lambda}{\lambda + \lambda} = \frac{1}{2}$。）

(b) $C$ 的总时间 = 等待时间 + 自己的服务时间。
- 等待时间 = $\min(T_A, T_B) \sim \text{Expo}(2\lambda)$（两个独立指数的最小值），期望 = $\frac{1}{2\lambda}$。
- 服务时间 $\sim \text{Expo}(\lambda)$，期望 = $\frac{1}{\lambda}$。

总期望 = $\frac{1}{2\lambda} + \frac{1}{\lambda} = \frac{3}{2\lambda}$。

**洞察**：这道题是理解指数分布无记忆性**实用价值**的经典例题。无记忆性让你可以在"第一个事件发生"的时刻**重置**分析——剩余时间的分布和刚开始时一样。这种"重置"能力是指数分布解题的核心优势。

**训练点**：无记忆性在排队论中的应用 | 独立指数最小值的分布 | 竞争性指数分布的概率

## 本章核心脉络（回顾）

### 第一层：两个核心概念——你必须理解的

1. **PDF 不是概率，是密度**：$f(x) > 1$ 完全合法。概率 = PDF × 区间长度（积分）。离散的点上没有概率——$P(X = c) = 0$ 对任何连续随机变量恒成立。
2. **CDF 是通用语言**：$F(x) = P(X \leq x)$ 在离散和连续中都有统一的定义。对连续变量，$F'(x) = f(x)$；对离散变量，$F$ 有跳跃。

### 第二层：三个核心分布——构成了连续世界的"基础构件"

- **均匀分布**：最简单，但它的"普适原理"使它能生成一切其他分布
- **正态分布**：CLT 的极限，测量的误差，自然界中无处不在的钟形
- **指数分布**：唯一有无记忆性的连续分布，泊松过程的等待时间

### 第三层：两个核心工具箱——解决实际问题的武器

- **LOTUS（连续版）**：$E(g(X)) = \int g(x) f_X(x) \, dx$——计算任何函数期望的统一方法
- **CDF 方法**：求变换后分布的标准流程——先写 $P(Y \leq y)$，然后反解出 $X$ 的范围，最后代入 $X$ 的分布

### 如果你只从这一章带走三样东西

1. **PDF ≠ 概率，PDF 是密度**。在连续分布中，任何单点的概率都是零。概率只有在区间上才存在。这一条如果记错了，后面所有联合连续分布的直觉都会跑偏。
2. **均匀性的普适原理**：$F(X) \sim \text{Unif}(0,1)$。这条定理连接了均匀分布和所有连续分布，是现代统计计算的基石。如果你看到 $\Phi(Z)$ 出现在期望里，不要 LOTUS——它直接就是 $\text{Unif}(0,1)$。
3. **无记忆性是稀有属性**：指数分布是**唯一**具有无记忆性的连续分布。当问题里有"等待"和"剩余时间"，第一时间看是否可以用指数分布建模，以及无记忆性能否帮你重置分析。
